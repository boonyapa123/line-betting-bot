# รูปแบบข้อความระบบการเล่น - Message Format Guide

## 📱 รูปแบบข้อความทั้งหมด

---

## 1️⃣ ข้อความยืนยันการเล่น (Confirmation Message)

### เมื่อผู้เล่นส่งข้อความเล่นสำเร็จ

**ตัวอย่าง:**
```
✅ บันทึกการเล่นสำเร็จ

ชื่อ: สมชาย
บั้งไฟ: บั้งไฟ1
ฝั่ง: ต
จำนวนเงิน: 500 บาท
```

**สำหรับ Reply Method:**
```
✅ บันทึกการเล่นสำเร็จ

ชื่อ: สมชาย
บั้งไฟ: บั้งไฟ1
ฝั่ง: ตอบ
```

**สำหรับ Direct Method (วิธีที่ 2):**
```
✅ บันทึกการเล่นสำเร็จ

ชื่อ: สมชาย
บั้งไฟ: บั้งไฟ1
ราคา: 40-50
ฝั่ง: ล
จำนวนเงิน: 500 บาท
```

---

## 2️⃣ ข้อความแจ้งผลแพ้ชนะ (Result Notification)

### ส่วนตัวผู้ชนะ

**ตัวอย่าง (ชนะ-แพ้):**
```
🎉 ยินดีด้วย! คุณชนะ

บั้งไฟ: บั้งไฟ1
คะแนนที่ออก: 50

เดิมพัน: 500 บาท
หัก: 50 บาท (10%)
ได้รับ: 450 บาท
```

**ตัวอย่าง (ออกกลาง):**
```
🎉 ยินดีด้วย! คุณชนะ

บั้งไฟ: บั้งไฟ1
คะแนนที่ออก: 50

🤝 ออกกลาง
หัก: 25 บาท (5%)
```

### ส่วนตัวผู้แพ้

**ตัวอย่าง (ชนะ-แพ้):**
```
😔 เสียใจด้วย คุณแพ้

บั้งไฟ: บั้งไฟ1
คะแนนที่ออก: 50

เดิมพัน: 500 บาท
```

**ตัวอย่าง (ออกกลาง):**
```
😔 เสียใจด้วย คุณแพ้

บั้งไฟ: บั้งไฟ1
คะแนนที่ออก: 50

🤝 ออกกลาง
หัก: 25 บาท (5%)
```

### ข้อความกลุ่ม

**ตัวอย่าง (ชนะ-แพ้):**
```
📊 ผลการเล่น บั้งไฟ: บั้งไฟ1
คะแนนที่ออก: 50
==================================================

🏆 ชนะ: สมชาย (สมชาย)
   ฝั่ง: ต
   เดิมพัน: 500 บาท
   หัก: 50 บาท (10%)
   ได้รับ: 450 บาท

❌ แพ้: สมหญิง (สมหญิง)
   ฝั่ง: ล
   เดิมพัน: 500 บาท

==================================================
```

**ตัวอย่าง (ออกกลาง):**
```
📊 ผลการเล่น บั้งไฟ: บั้งไฟ1
คะแนนที่ออก: 50
==================================================

🤝 ออกกลาง

👤 สมชาย (สมชาย)
   ฝั่ง: ต | เดิมพัน: 500 บาท
   หัก: 25 บาท (5%)

👤 สมหญิง (สมหญิง)
   ฝั่ง: ล | เดิมพัน: 500 บาท
   หัก: 25 บาท (5%)

==================================================
```

---

## 3️⃣ ข้อความสรุปยอดแทง (Summary Report)

### คำสั่ง: `:สรุป บั้งไฟ1 50`

**ตัวอย่าง:**
```
📊 สรุปผลการเล่น
บั้งไฟ: บั้งไฟ1
คะแนนที่ออก: 50
========================================

คู่ที่ 1:
🏆 ชนะ: สมชาย +450 บาท
❌ แพ้: สมหญิง -500 บาท

คู่ที่ 2:
🏆 ชนะ: สมศรี +450 บาท
❌ แพ้: สมปรารถนา -500 บาท

========================================
💰 ยอดเงินคงเหลือ:

สมชาย: 1450 บาท
สมหญิง: 500 บาท
สมศรี: 1450 บาท
สมปรารถนา: 500 บาท
```

---

## 4️⃣ ข้อความแจ้งเตือนยอดเงินไม่พอ (Insufficient Balance)

### เมื่อผู้เล่นเล่นแต่ยอดเงินไม่พอ

**ตัวอย่าง:**
```
❌ ยอดเงินไม่พอ

ยอดเงินปัจจุบัน: 300 บาท
จำนวนเงินที่ต้องการเดิมพัน: 500 บาท
ขาด: 200 บาท

💡 โปรดโอนเงินเพิ่มเติมและส่งสลิปให้ระบบตรวจสอบ
```

### ข้อความแจ้งเตือนส่วนตัว (ส่งอัตโนมัติ)

```
⚠️ ยอดเงินไม่พอสำหรับการเดิมพัน

ชื่อ: สมชาย
ยอดเงินปัจจุบัน: 300 บาท
จำนวนเงินที่ต้องการเดิมพัน: 500 บาท
ขาด: 200 บาท

💡 วิธีแก้ไข:
1. โอนเงินเพิ่มอย่างน้อย 200 บาท
2. ส่งสลิปการโอนเงินให้ระบบตรวจสอบ
3. รอการยืนยันจากระบบ
4. ลองเดิมพันใหม่อีกครั้ง

📱 ติดต่อแอดมิน หากมีปัญหา
```

---

## 5️⃣ ข้อความคำสั่งแอดมิน (Admin Commands)

### เริ่มรอบ
```
:เริ่ม บั้งไฟ1
```

**Response:**
```
✅ เปิดรอบการเล่น: บั้งไฟ1
รอบ ID: ROUND_1705756800000
```

### หยุดรอบ
```
:หยุด
```

**Response:**
```
✅ รอบนี้ปิดการทายแล้วคะ/ครับ
```

### สรุปผล
```
:สรุป บั้งไฟ1 50
```

**Response:**
```
📊 สรุปผลการเล่น
บั้งไฟ: บั้งไฟ1
คะแนนที่ออก: 50
========================================

คู่ที่ 1:
🏆 ชนะ: สมชาย +450 บาท
❌ แพ้: สมหญิง -500 บาท

========================================
💰 ยอดเงินคงเหลือ:

สมชาย: 1450 บาท
สมหญิง: 500 บาท
```

---

## 6️⃣ ข้อความข้อผิดพลาด (Error Messages)

### รูปแบบข้อความไม่ถูกต้อง
```
❌ รูปแบบข้อความไม่ถูกต้อง

💡 วิธีการเล่น:
- วิธีที่ 1: ต 500 (ฝั่ง จำนวนเงิน)
- วิธีที่ 2: 40-50 ล 500 (ราคา ฝั่ง จำนวนเงิน)
- Reply Method: ตอบ
```

### รอบปิดการทาย
```
❌ รอบนี้ปิดการทายแล้วคะ/ครับ
```

### ไม่พบชื่อบั้งไฟ
```
❌ ไม่พบชื่อบั้งไฟ กรุณารอให้แอดมินส่งชื่อบั้งไฟก่อน
```

### ไม่พบการเล่น
```
❌ ไม่พบการเล่นสำหรับบั้งไฟ: บั้งไฟ1
```

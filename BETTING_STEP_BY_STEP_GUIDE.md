# 🎰 Betting System - Step by Step Guide

## 📋 ขั้นตอนการทำงานทั้งหมด

---

## 🔵 ขั้นตอนที่ 1: Admin เปิดรอบ

### คำสั่ง
```
Admin: :เริ่ม ฟ้าหลังฝน
```

### ระบบทำอะไร

```
1. ตรวจจับข้อความ
   ├─ ค้นหา pattern ":เริ่ม"
   ├─ ดึงชื่อบั้งไฟ: "ฟ้าหลังฝน"
   └─ ประเภท: START command

2. ตรวจสอบความถูกต้อง
   ├─ ชื่อบั้งไฟไม่ว่าง ✅
   └─ สถานะปัจจุบัน: CLOSED ✅

3. บันทึกข้อมูล
   ├─ ชีท: RoundState
   ├─ State: OPEN
   ├─ RoundID: ROUND_1709443800000
   ├─ StartTime: 2024-03-02T10:30:00Z
   └─ SlipName: ฟ้าหลังฝน

4. ส่งข้อความตอบกลับ
   └─ ✅ เปิดรอบการเล่น: ฟ้าหลังฝน
      รอบ ID: ROUND_1709443800000
```

### ผลลัพธ์
```
✅ สถานะเปลี่ยนเป็น OPEN
✅ ระบบพร้อมรับการเล่น
✅ ผู้เล่นเห็นข้อความยืนยัน
```

## 🟢 ขั้นตอนที่ 2: User ส่งข้อความเล่น (วิธีที่ 1)

### คำสั่ง
```
Alice: ฟ้าหลังฝน ชล. 500
```

### ระบบทำอะไร

```
1. ตรวจจับข้อความ
   ├─ ใช้ Regex: ^(.+?)\s+(ชล\.|ชถ\.)\s+(\d+)$
   ├─ Group 1 (ชื่อบั้งไฟ): ฟ้าหลังฝน
   ├─ Group 2 (ฝั่ง): ชล.
   ├─ Group 3 (จำนวนเงิน): 500
   └─ ประเภท: Method 1

2. ตรวจสอบความถูกต้อง
   ├─ สถานะ: OPEN ✅
   ├─ จำนวนเงิน: 500 > 0 ✅
   ├─ จำนวนเงิน: 500 < 1000000 ✅
   └─ ชื่อบั้งไฟ: "ฟ้าหลังฝน" ✅

3. บันทึกข้อมูล
   ├─ ชีท: Bets
   ├─ Timestamp: 2024-03-02T10:30:00Z
   ├─ UserID: U001
   ├─ DisplayName: Alice
   ├─ Method: 1
   ├─ Price: (ว่าง)
   ├─ Side: ชล
   ├─ Amount: 500
   ├─ SlipName: ฟ้าหลังฝน
   └─ Status: OPEN

4. ส่งข้อความตอบกลับ
   └─ ✅ บันทึกการเล่นสำเร็จ
      ชื่อ: Alice
      บั้งไฟ: ฟ้าหลังฝน
      ฝั่ง: ไล่
      จำนวนเงิน: 500 บาท
```

### ผลลัพธ์
```
✅ ข้อมูลบันทึกลงชีท "Bets"
✅ Alice เห็นข้อความยืนยัน
✅ ระบบรอคู่ที่ตรงกัน
```

## 🟢 ขั้นตอนที่ 3: User ส่งข้อความเล่น (วิธีที่ 2)

### คำสั่ง
```
Bob: ฟ้าหลังฝน ชถ. 500
```

### ระบบทำอะไร

```
1. ตรวจจับข้อความ
   ├─ ใช้ Regex: ^(.+?)\s+(ชล\.|ชถ\.)\s+(\d+)$
   ├─ Group 1 (ชื่อบั้งไฟ): ฟ้าหลังฝน
   ├─ Group 2 (ฝั่ง): ชถ.
   ├─ Group 3 (จำนวนเงิน): 500
   └─ ประเภท: Method 1

2. ตรวจสอบความถูกต้อง
   ├─ สถานะ: OPEN ✅
   ├─ จำนวนเงิน: 500 > 0 ✅
   ├─ จำนวนเงิน: 500 < 1000000 ✅
   └─ ชื่อบั้งไฟ: "ฟ้าหลังฝน" ✅

3. บันทึกข้อมูล
   ├─ ชีท: Bets
   ├─ Timestamp: 2024-03-02T10:31:00Z
   ├─ UserID: U002
   ├─ DisplayName: Bob
   ├─ Method: 1
   ├─ Price: (ว่าง)
   ├─ Side: ชถ
   ├─ Amount: 500
   ├─ SlipName: ฟ้าหลังฝน
   └─ Status: OPEN

4. จับคู่การเล่น
   ├─ ค้นหาคู่ที่ตรงกัน
   ├─ Alice (ชล, 500, ฟ้าหลังฝน) ↔ Bob (ชถ, 500, ฟ้าหลังฝน)
   ├─ ตรวจสอบ:
   │  ├─ บั้งไฟเดียวกัน: ฟ้าหลังฝน = ฟ้าหลังฝน ✅
   │  ├─ จำนวนเงินเดียวกัน: 500 = 500 ✅
   │  ├─ ฝั่งตรงข้าม: ชล ↔ ชถ ✅
   │  └─ ผลลัพธ์: MATCHED ✅
   └─ บันทึก: คู่ที่ 1 = Alice ↔ Bob

5. ส่งข้อความตอบกลับ
   └─ ✅ บันทึกการเล่นสำเร็จ
      ชื่อ: Bob
      บั้งไฟ: ฟ้าหลังฝน
      ฝั่ง: ถอย
      จำนวนเงิน: 500 บาท
```

### ผลลัพธ์
```
✅ ข้อมูลบันทึกลงชีท "Bets"
✅ Bob เห็นข้อความยืนยัน
✅ ระบบจับคู่ Alice ↔ Bob สำเร็จ
```

## 🔴 ขั้นตอนที่ 4: Admin ปิดรอบ

### คำสั่ง
```
Admin: :หยุด
```

### ระบบทำอะไร

```
1. ตรวจจับข้อความ
   ├─ ค้นหา pattern ":หยุด"
   └─ ประเภท: STOP command

2. บันทึกข้อมูล
   ├─ ชีท: RoundState
   └─ State: CLOSED

3. ส่งข้อความตอบกลับ
   └─ รอบนี้ปิดการทายแล้วคะ/ครับ
```

### ผลลัพธ์
```
✅ สถานะเปลี่ยนเป็น CLOSED
✅ ระบบปฏิเสธการเล่นใหม่
✅ ผู้เล่นเห็นข้อความปิดรอบ
```

## 📊 ขั้นตอนที่ 5: Admin ประกาศผลลัพธ์

### คำสั่ง
```
Admin: :สรุป ฟ้าหลังฝน 315
```

### ระบบทำอะไร

```
1. ตรวจจับข้อความ
   ├─ ค้นหา pattern ":สรุป"
   ├─ ชื่อบั้งไฟ: ฟ้าหลังฝน
   ├─ คะแนน: 315
   └─ ประเภท: CALCULATE command

2. ดึงข้อมูลการเล่น
   ├─ ชีท: Bets
   ├─ ข้อมูล:
   │  ├─ Alice: ฟ้าหลังฝน, ชล, 500
   │  └─ Bob: ฟ้าหลังฝน, ชถ, 500
   └─ ผลลัพธ์: 2 คู่

3. จับคู่การเล่น
   ├─ คู่ที่ 1: Alice (ชล) ↔ Bob (ชถ)
   └─ ผลลัพธ์: 1 คู่ที่จับได้

4. คำนวณผลลัพธ์
   ├─ วิธีที่ 1: ฝั่ง "ไล่" (ชล) ชนะเสมอ
   ├─ Alice (ชล) = ไล่ → ชนะ ✅
   ├─ Bob (ชถ) = ถอย → แพ้ ✅
   └─ ผลลัพธ์:
      ├─ Alice: +500 บาท
      └─ Bob: -500 บาท

5. อัปเดตยอดเงิน
   ├─ ชีท: UsersBalance
   ├─ Alice: 1000 + 500 = 1500 บาท
   └─ Bob: 1000 - 500 = 500 บาท

6. ล้างข้อมูลการเล่น
   ├─ ชีท: Bets
   └─ ลบแถวทั้งหมด (A2:I)

7. ส่งข้อความตอบกลับ
   └─ 📊 สรุปผลการเล่น
      บั้งไฟ: ฟ้าหลังฝน
      คะแนนที่ออก: 315
      ========================================
      คู่ที่ 1:
      🏆 ชนะ: Alice +500 บาท
      ❌ แพ้: Bob -500 บาท
      ========================================
      💰 ยอดเงินคงเหลือ:
      Alice: 1500 บาท
      Bob: 500 บาท
```

### ผลลัพธ์
```
✅ ผลลัพธ์คำนวณสำเร็จ
✅ ยอดเงินอัปเดตแล้ว
✅ ข้อมูลการเล่นล้างแล้ว
✅ ผู้เล่นเห็นผลลัพธ์และยอดเงิน
```

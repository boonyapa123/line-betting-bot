# ระบบ LINE Betting Bot - การทำงานแบบสุดท้าย

## 📊 ภาพรวมการทำงาน (ขั้นตอนชัดเจน)

```
1. เติมเงิน (Deposit)
   ↓
2. เช็คยอดเงิน (Balance Check)
   ├─ ยอดเงินไม่พอ? → แจ้งเตือน → จบ
   └─ ยอดเงินพอ? ↓
3. จับคู่เล่น (Pair Matching)
   ├─ ไม่จับคู่ได้? → เก็บข้อความ → จบ
   └─ จับคู่ได้? ↓
4. บันทึกการเดิมพัน (Record Bet)
   ↓
5. ประกาศผล & แจ้งเตือน (Result & Notification)
```

---

## 1️⃣ ขั้นตอนเติมเงิน (Deposit)

### ผู้เล่นส่งสลิปการโอนเงิน
```
ผู้เล่น: ส่งรูปสลิป 500 บาท
        ↓
ระบบ: ตรวจสอบสลิป (Slip2Go API)
      ✅ สลิปถูกต้อง
        ↓
ระบบ: อัปเดตยอดเงิน
      - Players Sheet: ยอดเงิน += 500
      - Transactions Sheet: บันทึก (+500)
      - UsersBalance Sheet: อัปเดต
        ↓
ระบบ: ส่งแจ้งเตือน LINE ส่วนตัว
      "✅ ตรวจสอบสลิปสำเร็จ
       💰 เติมเงิน: 500 บาท
       💳 ยอดเงินใหม่: 5500 บาท
       🎉 พร้อมเล่นแล้ว!"
```

## 3️⃣ ขั้นตอนจับคู่เล่น (Pair Matching)

### เมื่อผู้เล่นคนที่ 2 ตอบกลับ

```

      1. ชื่อบั้งไฟตรงกัน? ✅
      2. ประเภทเล่นตรงข้าม? ✅
      3. ทั้งสองผู้เล่นลงทะเบียน? ✅
      4. ทั้งสองผู้เล่นยอดเงินพอ? ✅
```

## 5️⃣ ขั้นตอนประกาศผล & แจ้งเตือน (Result & Notification)

### ผู้ดำเนินการประกาศผล

```
ผู้ดำเนินการ: "ลูกชายภูน้อย ชล"
              ↓
ระบบ: ค้นหาการเดิมพันใน Bets Sheet
      ✅ พบการเดิมพัน
      ↓
ระบบ: อัปเดตผลลัพธ์
      - Column L (ผลลัพธ์ A): ชล (ชนะ)
      - Column M (ผลลัพธ์ B): ต (แพ้)
      ↓
ระบบ: อัปเดตยอดเงิน
      - User A: 5500 + 150 = 5650 บาท
      - User B: 4850 - 150 = 4700 บาท
      ↓
ระบบ: บันทึก Transactions Sheet
      - User A: ชนะเดิมพัน +150 บาท
      - User B: แพ้เดิมพัน -150 บาท
      ↓
ระบบ: ส่งแจ้งเตือน LINE ส่วนตัว
      
      User A (ชนะ):
      "🎉 ยินดีด้วย! คุณชนะ
       
       🎆 บั้งไฟ: ลูกชายภูน้อย
       🎯 ประเภท: ชล
       💰 ยอดเงิน: 150 บาท
       
       💵 ยอดเงินปัจจุบัน: 5650 บาท"
      
      User B (แพ้):
      "😢 ขออภัย คุณแพ้
       
       🎆 บั้งไฟ: ลูกชายภูน้อย
       🎯 ประเภท: ต
       💰 ยอดเงิน: 150 บาท
       
       💵 ยอดเงินปัจจุบัน: 4700 บาท"
      ↓
ระบบ: ส่งแจ้งเตือน กลุ่มแชท
      "✅ ประกาศผลแล้ว
       
       🎆 บั้งไฟ: ลูกชายภูน้อย
       🎯 ผลลัพธ์: ชล
       
       🏆 ผู้ชนะ:
       • 💓Noon💓 ชนะ 150 บาท
       
       😢 ผู้แพ้:
       • Pam Yuthida แพ้ 150 บาท"
      ↓
      → จบ
```

## 📝 สรุป

| ขั้นตอน | ทำอะไร | ผลลัพธ์ |
|--------|--------|--------|
| 1. เติมเงิน | อัปเดตยอดเงิน | ✅ เติมเงินสำเร็จ |
| 2. เช็คยอดเงิน | ตรวจสอบยอดเงิน | ✅ ยอดเงินพอ / ❌ ยอดเงินไม่พอ |
| 3. จับคู่เล่น | ตรวจสอบการจับคู่ | ✅ จับคู่ได้ / ❌ ไม่จับคู่ได้ |
| 4. บันทึกการเดิมพัน | บันทึกลง Bets Sheet | ✅ บันทึกสำเร็จ |
| 5. ประกาศผล | อัปเดตผลลัพธ์และยอดเงิน | ✅ ประกาศผลสำเร็จ |

### ประเด็นสำคัญ

✅ **ขั้นตอนที่ 2 (เช็คยอดเงิน)** - ถ้ายอดเงินไม่พอ → **จบที่นี่** (ไม่ไปขั้นตอนถัดไป)

✅ **ขั้นตอนที่ 3 (จับคู่เล่น)** - ถ้าไม่จับคู่ได้ → **จบที่นี่** (ไม่ไปขั้นตอนถัดไป)

✅ **ขั้นตอนที่ 4 (บันทึกการเดิมพัน)** - บันทึกลง Bets Sheet เท่านั้น

✅ **ขั้นตอนที่ 5 (ประกาศผล)** - อัปเดตยอดเงินและส่งแจ้งเตือน

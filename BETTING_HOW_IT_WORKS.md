# 🎰 Betting System - How It Works (สรุปการทำงาน)

## 📋 ตอบคำถามของคุณ

### ❓ บอททำงานอย่างไร?

**ตอบ:** ระบบทำงานตามลูป 5 ขั้นตอน:

```
1. Admin เปิดรอบ (:เริ่ม)
   ↓
2. User ส่งข้อความเล่น
   ↓
3. Admin ปิดรอบ (:หยุด)
   ↓
4. Admin ประกาศผล (:สรุป)
   ↓
5. ระบบประกาศผลลัพธ์เข้าห้องแชท
   ↓
(วนลูปไปเรื่อยๆ)
```

---

### ❓ ตรวจจับข้อความอย่างไร?

**ตอบ:** ใช้ **Regex Pattern** (นิพจน์ปกติ)

#### วิธีที่ 1 (ราคาช่าง)
```regex
^(.+?)\s+(ชล\.|ชถ\.)\s+(\d+)$
```

**ตัวอย่าง:**
```
ข้อความ: ฟ้าหลังฝน ชล. 500
         ↓
Group 1: ฟ้าหลังฝน (ชื่อบั้งไฟ)
Group 2: ชล. (ฝั่ง)
Group 3: 500 (จำนวนเงิน)
```

#### วิธีที่ 2 (ราคาคะแนน)
```regex
^(.+?)\s+([ลย]\.)\s+(\d+)\s+(.+)$
```

**ตัวอย่าง:**
```
ข้อความ: 0/3(300-330) ล. 500 ฟ้าหลังฝน
         ↓
Group 1: 0/3(300-330) (ราคา)
Group 2: ล. (ฝั่ง)
Group 3: 500 (จำนวนเงิน)
Group 4: ฟ้าหลังฝน (ชื่อบั้งไฟ)
```

#### คำสั่งแอดมิน
```
:เริ่ม [ชื่อบั้งไฟ]    → START
:หยุด                 → STOP
:สรุป [ชื่อบั้งไฟ] [คะแนน] → CALCULATE
```

### ❓ จับคู่เล่นอย่างไร?

**ตอบ:** ค้นหาคนที่เล่นบั้งไฟเดียวกัน ราคาเดียวกัน แต่ฝั่งตรงข้าม

```javascript
// ตรวจสอบ 4 เงื่อนไข
1. บั้งไฟเดียวกัน
   Alice: ฟ้าหลังฝน = Bob: ฟ้าหลังฝน ✅

2. จำนวนเงินเดียวกัน
   Alice: 500 = Bob: 500 ✅

3. ฝั่งตรงข้าม
   Alice: ชล (ไล่) ↔ Bob: ชถ (ถอย) ✅

4. ราคาเดียวกัน (วิธีที่ 2 เท่านั้น)
   Charlie: 0/3(300-330) = David: 0/3(300-330) ✅

ผลลัพธ์: MATCHED ✅
```

**ตัวอย่าง:**
```
ข้อมูลการเล่น:
1. Alice - ฟ้าหลังฝน ชล. 500
2. Bob - ฟ้าหลังฝน ชถ. 500
3. Charlie - พายุ ล. 1000 (0/3(300-330))
4. David - พายุ ย. 1000 (0/3(300-330))
5. Eve - เมฆา ล. 500 (0/3(300-330))

จับคู่ได้:
✅ คู่ที่ 1: Alice ↔ Bob
✅ คู่ที่ 2: Charlie ↔ David
⏳ Eve รอคู่ (Pending)
```

### ❓ บันทึกข้อมูลลงชีทอย่างไร?

**ตอบ:** ใช้ **Google Sheets API** บันทึกลงชีท "Bets"

```javascript
// ข้อมูลที่บันทึก
const row = [
  new Date().toISOString(),  // Timestamp: 2024-03-02T10:30:00Z
  userId,                     // UserID: U001
  displayName,                // DisplayName: Alice
  betData.method,             // Method: 1
  betData.price || '',        // Price: (ว่าง)
  betData.sideCode,           // Side: ชล
  betData.amount,             // Amount: 500
  betData.slipName,           // SlipName: ฟ้าหลังฝน
  'OPEN',                     // Status: OPEN
];

// บันทึกลงชีท
await sheets.spreadsheets.values.append({
  spreadsheetId: GOOGLE_SHEET_ID,
  range: 'Bets!A:I',
  valueInputOption: 'RAW',
  resource: { values: [row] },
});
```

**ตัวอย่างข้อมูลในชีท:**
```
Timestamp | UserID | DisplayName | Method | Price | Side | Amount | SlipName | Status
2024-03-02T10:30:00Z | U001 | Alice | 1 | | ชล | 500 | ฟ้าหลังฝน | OPEN
2024-03-02T10:31:00Z | U002 | Bob | 1 | | ชถ | 500 | ฟ้าหลังฝน | OPEN
```

### ❓ ประกาศให้ผู้เล่นทราบอย่างไร?

**ตอบ:** ส่งข้อความกลับไปยัง **LINE ห้องแชทเดียวกัน**

```javascript
// ส่งข้อความยืนยัน
const confirmationMessage = `✅ บันทึกการเล่นสำเร็จ

ชื่อ: Alice
บั้งไฟ: ฟ้าหลังฝน
ฝั่ง: ไล่
จำนวนเงิน: 500 บาท`;

// ส่งไปยัง LINE
await lineClient.replyMessage(replyToken, {
  type: 'text',
  text: confirmationMessage,
});
```

**ข้อความที่ส่ง:**
```
✅ บันทึกการเล่นสำเร็จ
ชื่อ: Alice
บั้งไฟ: ฟ้าหลังฝน
ฝั่ง: ไล่
จำนวนเงิน: 500 บาท
```

### ❓ มีแจ้งผลการเล่นเข้าห้องแชทนั้นๆด้วยไหม?

**ตอบ: ใช่ครับ! ✅**

ระบบจะส่งข้อความประกาศผลลัพธ์เข้าห้องแชทเดียวกัน

```javascript
// สร้างรายงานผลลัพธ์
const resultReport = `📊 สรุปผลการเล่น
บั้งไฟ: ฟ้าหลังฝน
คะแนนที่ออก: 315
========================================

คู่ที่ 1:
🏆 ชนะ: Alice +500 บาท
❌ แพ้: Bob -500 บาท

========================================
💰 ยอดเงินคงเหลือ:

Alice: 1500 บาท
Bob: 500 บาท`;

// ส่งไปยัง LINE
await lineClient.pushMessage(groupId, {
  type: 'text',
  text: resultReport,
});
```

**ข้อความที่ส่ง:**
```
📊 สรุปผลการเล่น
บั้งไฟ: ฟ้าหลังฝน
คะแนนที่ออก: 315
========================================

คู่ที่ 1:
🏆 ชนะ: Alice +500 บาท
❌ แพ้: Bob -500 บาท

========================================

Alice: 1500 บาท

```

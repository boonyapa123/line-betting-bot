# 📊 รูปแบบข้อมูลในแต่ละคอลัมน์ (Auto Matching)

## 📋 ตัวอย่างการบันทึก

### ตัวอย่าง: User A + User B จับคู่สำเร็จ

```
User A: "340-400 ล 300 แอด"
User B: "340-400 ย 300 แอด"
```

---

## 🔍 รายละเอียดข้อมูลในแต่ละคอลัมน์

### **Column A: Timestamp**
```
ค่า: 05/03/2569 02:49:23
รูปแบบ: DD/MM/YYYY HH:MM:SS (เวลาไทย)
ที่มา: เวลาที่ User A ส่งข้อความ
หมายเหตุ: ไม่เปลี่ยนแปลงเมื่ออัปเดต
```

### **Column B: User A ID**
```
ค่า: U51899d9b032327436b48ccb369a8505d
รูปแบบ: LINE User ID (ตัวอักษรและตัวเลข)
ที่มา: ดึงจาก source.userId ของ User A
หมายเหตุ: ไม่เปลี่ยนแปลงเมื่ออัปเดต
```

### **Column C: ชื่อ User A**
```
ค่า: ธา มือทอง
รูปแบบ: ชื่อ LINE (ตัวอักษรไทย/อังกฤษ)
ที่มา: ดึงจาก source.displayName ของ User A
หมายเหตุ: ไม่เปลี่ยนแปลงเมื่ออัปเดต
```

### **Column D: ข้อความ A**
```
ค่า: 340-400 ล 300 แอด
รูปแบบ: [ราคา] [ฝั่ง] [จำนวนเงิน] [ชื่อบั้งไฟ]
ที่มา: ข้อความที่ User A ส่งมา
หมายเหตุ: ไม่เปลี่ยนแปลงเมื่ออัปเดต
```

### **Column E: ชื่อบั้งไฟ**

### Step 1: User A ส่งข้อความ

```javascript
  price: "340-400",
  sideCode: "ล",
  amount: 300,
  slipName: "แอด"
}

// Create Row

  "05/03/2569 02:49:23",                    // A: Timestamp
  "U51899d9b032327436b48ccb369a8505d",     // B: User A ID
  "ธา มือทอง",                              // C: User A Name
  "340-400 ล 300 แอด",                      // D: Message A
  "แอด",                                    // E: Slip Name
  "ล",                                      // F: Side A
  300,                                      // G: Amount A
  "",                                       // H: Amount B (ว่างเปล่า)
  "",                                       // I: Result
  "",                                       // J: Result Win/Lose
  "",                                       // K: User B ID (ว่างเปล่า)

  "",                                       // M: Side B (ว่างเปล่า)
  "",                                       // N: Group Chat Name

  "",                                       // P: Token A
  "C4e522277480703e5eddbf658666ba6a9",     // Q: Group ID
  "",                                       // R: Token B (ว่างเปล่า)
  "",                                       // S: Result A
  "",                                       // T: Result B
  ""                                        // U: MATCHED Auto (ว่างเปล่า)

// Record to Sheet
await sheets.append(row)
```

| C | User A Name | ชื่อ | ❌ ไม่ |
| D | Message A | ข้อความ | ❌ ไม่ |
| E | Slip Name | ชื่อบั้งไฟ | ❌ ไม่ |
| F | Side A | ฝั่ง | ❌ ไม่ |
| G | Amount A | จำนวนเงิน | ❌ ไม่ |
| H | Amount B | จำนวนเงิน | ✅ อัปเดต |
| I | Result | ผลลัพธ์ | ✅ อัปเดต (ประกาศผล) |
| J | Result Win/Lose | ผลแพ้ชนะ | ✅ อัปเดต (ประกาศผล) |
| K | User B ID | ID | ✅ อัปเดต |
| L | User B Name | ชื่อ | ✅ อัปเดต |

| P | Token A | Token | ❌ ไม่ (ยังไม่มี) |
| Q | Group ID | ID | ❌ ไม่ |
| R | Token B | Token | ✅ อัปเดต (ยังไม่มี) |

# ระบบ LINE Betting Bot - การทำงานแบบปรับปรุง

## 📊 ภาพรวมการทำงาน (ปรับปรุง)

ระบบทำงานเป็น 5 ขั้นตอนหลัก:

```
1. เติมเงิน (Deposit)
   ↓
2. เช็คยอดเงิน (Balance Check)
   ↓
3. จับคู่เล่น (Pair Matching)
   ↓
4. บันทึกการเดิมพัน (Record Bet)
   ↓
5. ประกาศผล & แจ้งเตือน (Result & Notification)
```

---


## 1️⃣ ขั้นตอนเติมเงิน (Deposit) - ปรับปรุง

### ผู้เล่นส่งสลิปการโอนเงิน
- ผู้เล่นถ่ายรูปสลิปการโอนเงิน
- ส่งรูปไปยัง LINE Account 3 (Slip Verification Account)

### ระบบตรวจสอบสลิป
- **Slip2Go API** ตรวจสอบรูปสลิป
- ตรวจสอบ:
  - ชื่อผู้รับเงิน (ต้องตรงกับบัญชีที่ตั้งค่า)
  - จำนวนเงิน
  - วันที่โอน
  - ธนาคารต้นทาง

### อัปเดตยอดเงิน
- ถ้าสลิปถูกต้อง → อัปเดต **Players sheet** (ยอดเงินปัจจุบัน)
- บันทึกลง **Transactions sheet** (บันทึกการโอนเงิน)
- อัปเดต **UsersBalance sheet** (ยอดเงินปัจจุบัน)

### ส่งแจ้งเตือน
- ส่งข้อความไปยัง LINE ส่วนตัวของผู้เล่น
- ข้อความ: "✅ ยืนยันการเติมเงิน [จำนวน] บาท สำเร็จ"

### 🔄 **ใหม่: Re-process Stored Messages**
```
ระบบตรวจสอบ messageMap:
  - มีข้อความแทงที่เก็บไว้จากผู้เล่นนี้หรือไม่?
  
ถ้ามี:
  1. ดึงข้อความแทงที่เก็บไว้
  2. ตรวจสอบยอดเงินใหม่
  3. ถ้ายอดเงินพอแล้ว:
     → ส่งข้อความยืนยัน
     "✅ ยอดเงินเพียงพอแล้ว
      ข้อความแทง: "ลูกชายภูน้อย ชล 150"
      ยอดเงินปัจจุบัน: 500 บาท
      ต้องการ: 150 บาท
      
      🎉 พร้อมเล่นแล้ว!"
```

## 2️⃣ ขั้นตอนเช็คยอดเงิน (Balance Check)

### เมื่อผู้เล่นส่งข้อความแทง

**ตัวอย่าง:** "ลูกชายภูน้อย ชล 150"

### ระบบทำการ:

#### 1. ดึงข้อมูลผู้เล่น
```
- ชื่อ LINE: ดึงจาก LINE Profile
- User ID: ดึงจาก LINE API
```

#### 2. ค้นหาผู้เล่นในระบบ
```
ค้นหาลำดับ:
1️⃣  ค้นหาจากชื่อ LINE (Players sheet, column B)
2️⃣  ค้นหาจาก Linked IDs (Players sheet, column C)
3️⃣  ค้นหาจาก User ID (Players sheet, column A)
```

#### 3. ตรวจสอบสถานะผู้เล่น

**กรณี 1: ผู้เล่นไม่ลงทะเบียน**
```
❌ ยังไม่ลงทะเบียนในระบบ

ชื่อ: [ชื่อผู้เล่น]
ข้อความแทง: "ลูกชายภูน้อย ชล 150"

💡 วิธีแก้ไข:
1️⃣  ติดต่อแอดมิน
2️⃣  ให้แอดมินเพิ่มชื่อของคุณในระบบ
3️⃣  ลองเดิมพันใหม่อีกครั้ง

📞 ติดต่อสอบถาม: @774pojob
```
→ ส่งไปยัง **LINE ส่วนตัว** เท่านั้น

**กรณี 2: ผู้เล่นลงทะเบียนแล้ว แต่ยอดเงินไม่พอ**
```
❌ ยอดเงินไม่เพียงพอ

ชื่อ: [ชื่อผู้เล่น]
ข้อความแทง: "ลูกชายภูน้อย ชล 150"
ยอดเงินปัจจุบัน: 100 บาท
ต้องการ: 150 บาท
ขาดอีก: 50 บาท

💳 ช่องทางเติมเงิน:
• เพิ่มเพื่อน @774pojob
• https://lin.ee/JO6X7FE

📞 ติดต่อสอบถาม: @774pojob
```
→ ส่งไปยัง **LINE ส่วนตัว** เท่านั้น
→ 📦 **เก็บข้อความไว้ใน messageMap** เพื่อรอการเติมเงิน

**กรณี 3: ผู้เล่นลงทะเบียนแล้ว และยอดเงินพอ**
```
✅ ยอดเงินเพียงพอ
→ ไปยังขั้นตอนถัดไป (จับคู่เล่น)
```

## 3️⃣ ขั้นตอนจับคู่เล่น (Pair Matching)

### วิธีการจับคู่ 2 แบบ

#### วิธี 1: Reply Method (ตอบกลับข้อความ)
```
User A: "ลูกชายภูน้อย ชล 150"
User B: ตอบกลับข้อความของ User A ด้วย "ต"
```

#### วิธี 2: Direct Method (ส่งข้อความตรง)
```
User A: "ลูกชายภูน้อย ชล 150"
User B: "ลูกชายภูน้อย ต 150"
```

### ระบบตรวจสอบการจับคู่

#### 1. ตรวจสอบชื่อบั้งไฟ
```
User A: "ลูกชายภูน้อย ชล 150"
User B: "ลูกชายภูน้อย ต 150"
→ ✅ ชื่อบั้งไฟตรงกัน
```

#### 2. ตรวจสอบประเภทเล่น
```
User A: ชล (ต่ำ)
User B: ต (สูง)
→ ✅ ประเภทตรงข้ามกัน
```

#### 3. ตรวจสอบยอดเงิน (ทั้ง User A และ User B)

**ถ้าผู้เล่นคนใดคนหนึ่งไม่ลงทะเบียน:**
```
⚠️ ⚠️ ⚠️ ผู้เล่นยังไม่ลงทะเบียน ⚠️ ⚠️ ⚠️

👤 [ชื่อผู้เล่น] ยังไม่ลงทะเบียน

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💡 วิธีแก้ไข:
1️⃣  ติดต่อแอดมิน
2️⃣  ให้แอดมินเพิ่มชื่อในระบบ
3️⃣  ลองเดิมพันใหม่อีกครั้ง

📱 ติดต่อแอดมิน หากมีปัญหา
```
→ ส่งไปยัง **LINE ส่วนตัว** ของผู้เล่นที่ไม่ลงทะเบียน
→ ส่งไปยัง **กลุ่มแชท** ด้วย

**ถ้าผู้เล่นลงทะเบียนแล้ว แต่ยอดเงินไม่พอ:**
```
⚠️ ⚠️ ⚠️ ยอดเงินไม่เพียงพอ ⚠️ ⚠️ ⚠️

👤 [ชื่อผู้เล่น A] ยอดเงินไม่พอ (ขาด 50 บาท)
👤 [ชื่อผู้เล่น B] ยอดเงินไม่พอ (ขาด 100 บาท)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💡 วิธีแก้ไข:
1️⃣  โอนเงินเพิ่มเติมให้เพียงพอ
2️⃣  ส่งสลิปการโอนให้ระบบตรวจสอบ
3️⃣  รอการยืนยันจากระบบ
4️⃣  ลองเดิมพันใหม่อีกครั้ง

📱 ติดต่อแอดมิน หากมีปัญหา
```
→ ส่งไปยัง **LINE ส่วนตัว** ของผู้เล่นที่ยอดเงินไม่พอ
→ ส่งไปยัง **กลุ่มแชท** ด้วย

**ถ้าทั้งสองผู้เล่นลงทะเบียนและยอดเงินพอ:**
```
✅ ทั้งสองผู้เล่นพร้อมเล่น
→ ไปยังขั้นตอนถัดไป (บันทึกการเดิมพัน)
```

## 5️⃣ ขั้นตอนประกาศผล & แจ้งเตือน (Result & Notification)

### ผู้ดำเนินการประกาศผล

**ตัวอย่าง:** "ลูกชายภูน้อย ชล"

### ระบบทำการ:

#### 1. ค้นหาการเดิมพันที่ตรงกัน
```
ค้นหาจาก Bets Sheet:
- ชื่อบั้งไฟ = "ลูกชายภูน้อย"
- ยังไม่มีผลลัพธ์
```

#### 2. อัปเดตผลลัพธ์
```
Bets Sheet:
- Column L (ผลลัพธ์ A): ชล (ชนะ)
- Column M (ผลลัพธ์ B): ต (แพ้)
```

#### 3. อัปเดตยอดเงิน

**User A (ชนะ):**
```
ยอดเงินเดิม: 5000 บาท
+ ชนะ: 150 บาท
= ยอดเงินใหม่: 5150 บาท
```

**User B (แพ้):**
```
ยอดเงินเดิม: 5000 บาท
- แพ้: 150 บาท
= ยอดเงินใหม่: 4850 บาท
```

#### 4. บันทึก Transactions Sheet
```
| ผู้เล่น | ประเภท | จำนวน | ยอดเงินใหม่ | หมายเหตุ |
|--------|--------|--------|------------|---------|
| 💓Noon💓 | ชนะเดิมพัน | +150 | 5150 | ลูกชายภูน้อย ชล |
| Pam Yuthida | แพ้เดิมพัน | -150 | 4850 | ลูกชายภูน้อย ต |
```

#### 5. ส่งแจ้งเตือน

**ส่งไปยัง LINE ส่วนตัว:**

**User A (ชนะ):**
```

🎆 บั้งไฟ: ลูกชายภูน้อย
🎯 ประเภท: ชล
💰 ยอดเงิน: 150 บาท

💵 ยอดเงินปัจจุบัน: 5150 บาท
```

**User B (แพ้):**

**ส่งไปยัง กลุ่มแชท:**

## 📝 สรุปการปรับปรุง

| ขั้นตอน | เดิม | ปรับปรุง |
|--------|------|---------|

✅ **ลดการสับสน**
- ผู้เล่นไม่ต้องจำว่าต้องส่งข้อความแทงใหม่

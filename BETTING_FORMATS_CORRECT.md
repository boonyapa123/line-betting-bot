# 🎰 รูปแบบการเล่นที่ถูกต้อง - Correct Betting Formats

## 📋 ระบบการเล่น 2 วิธี

ระบบรองรับการเล่น 2 วิธี:

---

## 🔵 วิธีที่ 1: Direct + Reply Method (ลงเล่นตรง + ตอบกลับ)

### วิธีการเล่น

```
User A: ฟ้าหลังฝน ชล. 500 (ลงเล่นตรง)
       ↓
User B: ต (reply ข้อความของ User A)
       ↓
ระบบจับคู่ User A ↔ User B ✅
(User B อยู่ฝั่งตรงข้าม = ชถ)
```

### ตัวอย่าง

```
┌─────────────────────────────────────────┐
│ User A: ฟ้าหลังฝน ชล. 500              │
│ → ระบบบันทึก: User A เล่น ชล 500      │
│ → สถานะ: OPEN (รอคู่)                  │
└─────────────────────────────────────────┘

┌─────────────────────────────────────────┐
│ User B: ต (reply ข้อความของ User A)    │
│ → ระบบบันทึก: User B เล่น ชถ (ตรงข้าม)│
│ → ระบบจับคู่: User A ↔ User B ✅       │
│ → สถานะ: MATCHED                       │
└─────────────────────────────────────────┘
```

### ข้อมูลที่บันทึก

```
User A:
  - SlipName: ฟ้าหลังฝน
  - Side: ชล (ไล่)
  - Amount: 500
  - Status: OPEN → MATCHED

User B:
  - SlipName: ฟ้าหลังฝน
  - Side: ชถ (ถอย) ← ตรงข้ามกับ User A
  - Amount: 500 ← เท่ากับ User A
  - Status: OPEN → MATCHED
```

### ตรรกะการจับคู่

```javascript
// ตรวจสอบ 3 เงื่อนไข
1. ชื่อบั้งไฟเดียวกัน
   User A: ฟ้าหลังฝน = User B: ฟ้าหลังฝน ✅

2. จำนวนเงินเดียวกัน
   User A: 500 = User B: 500 ✅

3. ฝั่งตรงข้าม (User B reply = ฝั่งตรงข้าม)
   User A: ชล (ไล่) ↔ User B: ชถ (ถอย) ✅

ผลลัพธ์: MATCHED ✅
```

---

## 🟢 วิธีที่ 2: Direct + Direct Method (ลงเล่นตรง + ลงเล่นตรง)

### วิธีการเล่น

```
User A: ฟ้าหลังฝน ชล. 500 (ลงเล่นตรง)
       ↓
User B: ฟ้าหลังฝน ชถ. 300 (ลงเล่นตรง, ฝั่งตรงข้าม)
       ↓
ระบบจับคู่ User A ↔ User B ✅
(ใช้ยอดน้อยกว่า = 300 บาท)
```

### ตัวอย่าง

```
┌─────────────────────────────────────────┐
│ User A: ฟ้าหลังฝน ชล. 500              │
│ → ระบบบันทึก: User A เล่น ชล 500      │
│ → สถานะ: OPEN (รอคู่)                  │
└─────────────────────────────────────────┘

┌─────────────────────────────────────────┐
│ User B: ฟ้าหลังฝน ชถ. 300              │
│ → ระบบบันทึก: User B เล่น ชถ 300      │
│ → ระบบจับคู่: User A ↔ User B ✅       │
│ → ใช้ยอดน้อยกว่า: 300 บาท             │
│ → สถานะ: MATCHED                       │
└─────────────────────────────────────────┘
```

### ข้อมูลที่บันทึก

```
User A:
  - SlipName: ฟ้าหลังฝน
  - Side: ชล (ไล่)
  - Amount: 500
  - Status: OPEN → MATCHED

User B:
  - SlipName: ฟ้าหลังฝน
  - Side: ชถ (ถอย)
  - Amount: 300
  - Status: OPEN → MATCHED

ยอดเดิมพันที่ใช้: 300 บาท (ยอดน้อยกว่า)
```

### ตรรกะการจับคู่

```javascript
// ตรวจสอบ 3 เงื่อนไข
1. ชื่อบั้งไฟเดียวกัน
   User A: ฟ้าหลังฝน = User B: ฟ้าหลังฝน ✅

2. ฝั่งตรงข้าม
   User A: ชล (ไล่) ↔ User B: ชถ (ถอย) ✅

3. ราคาเดียวกัน (วิธีที่ 2 เท่านั้น)
   User A: (ไม่มี) = User B: (ไม่มี) ✅

ผลลัพธ์: MATCHED ✅
ยอดเดิมพัน: min(500, 300) = 300 บาท
```

## 📊 ตัวอย่างการทำงานทั้งรอบ

### Scenario 1: Direct + Reply Method

```
1️⃣ User A: ฟ้าหลังฝน ชล. 500
   → Bets: User A, ฟ้าหลังฝน, ชล, 500, OPEN

2️⃣ User B: ต (reply ข้อความของ User A)
   → Bets: User B, ฟ้าหลังฝน, ชถ, 500, OPEN
   → จับคู่: User A ↔ User B ✅

3️⃣ Admin: :หยุด
   → RoundState: CLOSED

4️⃣ Admin: :สรุป ฟ้าหลังฝน 315
   → คำนวณผล
   → ประกาศผล
```

### Scenario 2: Direct + Direct Method

```
1️⃣ User A: ฟ้าหลังฝน ชล. 500
   → Bets: User A, ฟ้าหลังฝน, ชล, 500, OPEN

2️⃣ User B: ฟ้าหลังฝน ชถ. 500
   → Bets: User B, ฟ้าหลังฝน, ชถ, 500, OPEN
   → จับคู่: User A ↔ User B ✅

3️⃣ Admin: :หยุด
   → RoundState: CLOSED

4️⃣ Admin: :สรุป ฟ้าหลังฝน 315
   → คำนวณผล
   → ประกาศผล
```

| ลักษณะ | วิธีที่ 1 (Direct + Reply) | วิธีที่ 2 (Direct + Direct) |
|--------|--------------------------|---------------------------|
| User A | ฟ้าหลังฝน ชล. 500 | ฟ้าหลังฝน ชล. 500 |
| User B | ต (reply) | ฟ้าหลังฝน ชถ. 500 |
| ตรวจจับ | Reply pattern | Regex pattern |
| ฝั่ง User B | ตรงข้ามอัตโนมัติ | ระบุเอง |
| ความเร็ว | เร็ว | ปกติ |
| ความง่าย | ง่าย | ปกติ |

## 🔍 ตรวจจับข้อความ

### วิธีที่ 1 (Direct + Reply)

**User A (Direct):**
```regex
^(.+?)\s+(ชล\.|ชถ\.)\s+(\d+)$
```

**User B (Reply):**
```
ต หรือ ล หรือ ย
```

### วิธีที่ 2 (Direct + Direct)

**User A (Direct):**
```regex
^(.+?)\s+(ชล\.|ชถ\.)\s+(\d+)$
```

**User B (Direct):**
```regex
^(.+?)\s+(ชล\.|ชถ\.)\s+(\d+)$
```

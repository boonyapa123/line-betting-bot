# ระบบแจ้งเตือนการจับคู่อัตโนมัติ

## 📋 ภาพรวม

เมื่อระบบจับคู่เล่นได้สำเร็จ จะมีการแจ้งเตือนไปยัง:
1. ✅ ข้อความส่วนตัวให้ผู้เล่น A
2. ✅ ข้อความส่วนตัวให้ผู้เล่น B
3. ✅ ข้อความเข้ากลุ่ม LINE (ถ้ามี)

---

## 🎯 ตัวอย่างการแจ้งเตือน

### ข้อความส่วนตัวผู้เล่น A (ผู้เล่นใหม่)

```
✅ จับคู่เล่นสำเร็จ (ร้องราคา)

👤 คุณ: [ชื่อผู้เล่น A]
👤 คู่แข่ง: [ชื่อผู้เล่น B]
🎆 บั้งไฟ: ฟ้าหลังฝน
💹 ราคาของคุณ: 300-330
💹 ราคาคู่แข่ง: 350-380
💰 ยอดเงิน: 300 บาท

⏳ รอการประกาศผล...
```

### ข้อความส่วนตัวผู้เล่น B (ผู้เล่นเดิม)

```
✅ จับคู่เล่นสำเร็จ (ร้องราคา)

👤 คุณ: [ชื่อผู้เล่น B]
👤 คู่แข่ง: [ชื่อผู้เล่น A]
🎆 บั้งไฟ: ฟ้าหลังฝน
💹 ราคาของคุณ: 350-380
💹 ราคาคู่แข่ง: 300-330
💰 ยอดเงิน: 300 บาท

⏳ รอการประกาศผล...
```

### ข้อความกลุ่ม LINE

```
✅ จับคู่เล่นสำเร็จ (ร้องราคา)

👤 [ชื่อผู้เล่น A] vs [ชื่อผู้เล่น B]
🎆 บั้งไฟ: ฟ้าหลังฝน
💹 ราคา A: 300-330
💹 ราคา B: 350-380
💰 ยอดเงิน: 300 บาท

⏳ รอการประกาศผล...
```

---

## 🔧 โครงสร้างโค้ด

### ไฟล์ที่เกี่ยวข้อง

1. **bettingRoundController.js**
   - ตรวจสอบการจับคู่อัตโนมัติ
   - สร้างข้อความแจ้งเตือน
   - ส่งข้อความไปยัง LINE

2. **priceRangeMatchingService.js**
   - `createAutoMatchMessage()` - สร้างข้อความแจ้งการจับคู่

3. **lineNotificationService.js**
   - `sendPrivateMessage()` - ส่งข้อความส่วนตัว
   - `sendGroupMessage()` - ส่งข้อความกลุ่ม

## 📊 ขั้นตอนการแจ้งเตือน

```
User B: 350-380 ย 300 ฟ้าหลังฝน
    ↓
[Parse Message] → method: 2, side: ย, price: 350-380, amount: 300
    ↓
[Record Bet] → บันทึกลงฐานข้อมูล
    ↓
[Check Balance] → ตรวจสอบยอดเงิน ✅
    ↓
[Find Match] → ค้นหาคู่ที่มีฝั่งตรงข้าม
    ↓
    ✅ พบ User A → จับคู่อัตโนมัติ
    ↓
[Create Messages] → สร้างข้อความแจ้งเตือน
    ├─ messageA: ข้อความสำหรับ User A
    ├─ messageB: ข้อความสำหรับ User B
    └─ groupMessage: ข้อความสำหรับกลุ่ม
    ↓
[Send Notifications]
    ├─ 📤 ส่งข้อความส่วนตัวให้ User A
    ├─ 📤 ส่งข้อความส่วนตัวให้ User B
    └─ 📢 ส่งข้อความเข้ากลุ่ม (ถ้ามี groupId)
    ↓
✅ แจ้งเตือนสำเร็จ
```

## 🚀 ฟีเจอร์

### ที่ได้ปรับปรุง

1. ✅ ส่งข้อความส่วนตัวให้ผู้เล่น A
2. ✅ ส่งข้อความส่วนตัวให้ผู้เล่น B
3. ✅ ส่งข้อความเข้ากลุ่ม LINE
4. ✅ แสดงราคาของทั้งสองฝั่ง (ถ้าต่างกัน)
5. ✅ แสดงยอดเงินที่ใช้ (ยอดน้อยกว่า)

### ที่ยังต้องปรับปรุง

- [ ] เพิ่มการส่งข้อความแบบ Flex Message
- [ ] เพิ่มการส่งข้อความแบบ Rich Menu
- [ ] เพิ่มการบันทึกประวัติการแจ้งเตือน
- [ ] เพิ่มการตรวจสอบสถานะการส่งข้อความ

---

## 📝 หมายเหตุ

- ใช้ Account 1 สำหรับการส่งข้อความแจ้งเตือน
- ส่งข้อความส่วนตัวก่อน จากนั้นส่งข้อความกลุ่ม
- ถ้าส่งข้อความไม่สำเร็จ จะบันทึก error แต่ไม่หยุดการทำงาน
- ข้อความแจ้งเตือนจะแสดงราคาของทั้งสองฝั่ง (ถ้าต่างกัน)

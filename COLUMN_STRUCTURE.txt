================================================================================
GOOGLE SHEETS COLUMN STRUCTURE - CURRENT
================================================================================

📋 BETS SHEET (ชีท Bets)
================================================================================

Row 1 (Header - เขียนเพียงครั้งเดียว):
  A1: Timestamp
  B1: User A ID
  C1: ชื่อ User A
  D1: ข้อความ A
  E1: ชื่อบั้งไฟ
  F1: รายการเล่น
  G1: ยอดเงิน A
  H1: ยอดเงิน B
  I1: แสดงผล
  J1: แสดงผลชนะ
  K1: User B ID
  L1: ชื่อ User B
  M1: รายการเล่น B
  N1: ผลลัพธ์สุดท้าย

Row 2 onwards (Data - ระบบเขียนอัตโนมัติ):
  A: 2024-03-04T10:30:00.000Z
  B: U001
  C: John
  D: ลูกชายภูน้อย ชล 150
  E: ลูกชายภูน้อย
  F: ชล
  G: 150
  H: (ว่างเปล่า - รอการจับคู่)
  I: (ว่างเปล่า - รอผลลัพธ์)
  J: (ว่างเปล่า)
  K: (ว่างเปล่า - รอการจับคู่)
  L: (ว่างเปล่า - รอการจับคู่)
  M: (ว่างเปล่า - รอการจับคู่)
  N: (ว่างเปล่า - รอผลลัพธ์)

================================================================================

💰 USERSBALANCE SHEET (ชีท UsersBalance)
================================================================================

Row 1 (Header - เขียนเพียงครั้งเดียว):
  A1: User ID
  B1: Display Name
  C1: Balance

Row 2 onwards (Data - ข้อมูลผู้เล่น):
  A: U001
  B: John
  C: 1000

  A: U002
  B: Jane
  C: 800

================================================================================

📝 TRANSACTIONS SHEET (ชีท Transactions)
================================================================================

Row 1 (Header - เขียนเพียงครั้งเดียว):
  A1: Timestamp
  B1: Player Name
  C1: Transaction Type
  D1: Amount
  E1: Previous Balance
  F1: New Balance
  G1: Slip Name

Row 2 onwards (Data - ระบบเขียนอัตโนมัติ):
  A: 2024-03-04T10:35:00.000Z
  B: John
  C: WIN
  D: 270
  E: 700
  F: 970
  G: ลูกชายภูน้อย

  A: 2024-03-04T10:35:00.000Z
  B: Jane
  C: LOSE
  D: -300
  E: 500
  F: 500
  G: ลูกชายภูน้อย

================================================================================

✅ SETUP INSTRUCTIONS
================================================================================

1. สร้างชีท 3 ชีท:
   - Bets
   - UsersBalance
   - Transactions

2. เขียนชื่อคอลัมน์ใน Row 1 (ตามด้านบน)

3. เพิ่มข้อมูลผู้เล่นใน UsersBalance Sheet (Row 2 onwards)

4. ระบบจะเขียนข้อมูลอัตโนมัติ ไม่ต้องเขียนชื่อคอลัมน์ใหม่

================================================================================

⚠️ IMPORTANT NOTES
================================================================================

1. ไม่ต้องเขียนชื่อคอลัมน์ทุกครั้ง
   - เขียนเพียงครั้งเดียวใน Row 1
   - ระบบจะเขียนข้อมูลใน Row 2 onwards

2. ไม่ต้องสร้างแถวใหม่
   - ระบบจะสร้างแถวใหม่อัตโนมัติ

3. ไม่ต้องคำนวนเงิน
   - ระบบจะคำนวนอัตโนมัติ

4. ไม่ต้องอัปเดตยอดเงิน
   - ระบบจะอัปเดตอัตโนมัติ

5. ไม่ต้องเขียนผลลัพธ์
   - ระบบจะเขียนผลลัพธ์อัตโนมัติ

================================================================================

🔄 DATA FLOW
================================================================================

Step 1: ผู้เล่นเดิมพัน
  → Bets Sheet Row 2: [Timestamp, User A ID, ชื่อ User A, ข้อความ, ชื่อบั้งไฟ, รายการเล่น, ยอดเงิน, '', '', '', '', '', '', '']

Step 2: จับคู่สำเร็จ
  → Bets Sheet Row 2: [Timestamp, User A ID, ชื่อ User A, ข้อความ, ชื่อบั้งไฟ, รายการเล่น, ยอดเงิน A, ยอดเงิน B, '', '', User B ID, ชื่อ User B, รายการเล่น B, '']
  → UsersBalance Sheet: Update User A Balance: 1000 - 300 = 700
  → UsersBalance Sheet: Update User B Balance: 800 - 300 = 500

Step 3: ออกผลลัพธ์
  → Bets Sheet Row 2: [Timestamp, User A ID, ชื่อ User A, ข้อความ, ชื่อบั้งไฟ, รายการเล่น, ยอดเงิน A, ยอดเงิน B, ผลลัพธ์, ผลลัพธ์ชนะ, User B ID, ชื่อ User B, รายการเล่น B, ผลลัพธ์สุดท้าย]
  → UsersBalance Sheet: Update User A Balance: 700 + 270 = 970 (ชนะ)
  → UsersBalance Sheet: Update User B Balance: 500 (เสีย)
  → Transactions Sheet Row 2: [Timestamp, John, WIN, 270, 700, 970, ลูกชายภูน้อย]
  → Transactions Sheet Row 3: [Timestamp, Jane, LOSE, -300, 500, 500, ลูกชายภูน้อย]

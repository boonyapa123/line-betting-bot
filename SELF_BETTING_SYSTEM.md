# ระบบการเล่นกับตัวเอง - Self-Betting System

## 📋 ภาพรวม

ระบบได้รับการปรับปรุงให้ผู้เล่นสามารถ **เล่นกับตัวเองได้** โดยการเล่นมากกว่า 1 ครั้งในรอบเดียว

---

## 🔧 ไฟล์ที่เพิ่มเติม

### `services/betting/selfBettingService.js`
บริหารจัดการการเล่นกับตัวเอง

**ฟังก์ชันหลัก:**

#### `findSelfBets(bets, slipName)`
- ค้นหาผู้เล่นที่เล่นมากกว่า 1 ครั้ง
- **Return:** Array ของผู้เล่นที่เล่นกับตัวเอง

#### `calculateSelfBettingResults(selfBet, slipName, score)`
- คำนวณผลลัพธ์การเล่นกับตัวเอง
- จับคู่การเล่นของผู้เล่นคนเดียว
- **Return:** Array ของผลลัพธ์

#### `buildSelfBettingMessage(result, slipName, score)`
- สร้างข้อความแจ้งเตือนสำหรับการเล่นกับตัวเอง
- **Return:** String ข้อความ

#### `validateSelfBetting(bets)`
- ตรวจสอบว่าการเล่นกับตัวเองถูกต้องหรือไม่
- **Return:** Object {valid, issues}

## 💰 การคำนวณยอดเงิน

### การเล่นกับตัวเอง

**ก่อนเล่น:**
```
สมชาย: 1000 บาท
```

**เล่น:**
- ฝั่ง ต: 500 บาท
- ฝั่ง ล: 500 บาท

**ผลลัพธ์ (คะแนน 50 ออก ฝั่ง ต ชนะ):**
```
ฝั่ง ต ชนะ: +500 บาท
ค่าธรรมเนียม: -50 บาท (10%)
ได้รับสุทธิ: +450 บาท

ฝั่ง ล แพ้: -500 บาท
```

**หลังเล่น:**
```
สมชาย: 1000 + 450 - 500 = 950 บาท
```

## ✅ ตรวจสอบการเล่นกับตัวเอง

### เงื่อนไขการเล่นกับตัวเอง

1. **ต้องมีการเล่นอย่างน้อย 2 ครั้ง**
   ```
   ✅ ผู้เล่นเล่น 2 ครั้ง
   ❌ ผู้เล่นเล่น 1 ครั้ง
   ```

2. **ทุกการเล่นต้องเป็นของคนเดียวกัน**
   ```
   ✅ สมชาย เล่น 2 ครั้ง
   ❌ สมชาย เล่น 1 ครั้ง + สมหญิง เล่น 1 ครั้ง
   ```

3. **ต้องมีฝั่งตรงข้ามอย่างน้อย 2 ฝั่ง**
   ```
   ✅ ฝั่ง ต + ฝั่ง ล
   ❌ ฝั่ง ต + ฝั่ง ต
   ```

## 📊 ตัวอย่างการสรุปผล (Mixed)

### Scenario: ผู้เล่น 4 คน (2 คนเล่นปกติ + 1 คนเล่นกับตัวเอง)

#### ข้อมูลการเล่น
```
สมชาย: ต 500
สมหญิง: ล 500
สมศรี: ต 300
สมศรี: ล 300
```

#### ผลลัพธ์
```
คู่ที่ 1 (ปกติ):
🏆 ชนะ: สมชาย (ต) +450 บาท
❌ แพ้: สมหญิง (ล) -500 บาท

คู่ที่ 2 (กับตัวเอง):
📊 ผลการเล่นกับตัวเอง
🏆 ฝั่ง ต ชนะ: +270 บาท
❌ ฝั่ง ล แพ้: -300 บาท
```

#### ยอดเงินหลังเล่น
```
สมชาย: 1000 + 450 = 1450 บาท
สมหญิง: 1000 - 500 = 500 บาท
สมศรี: 1000 + 270 - 300 = 970 บาท
```

## 🎯 ข้อดีของการเล่นกับตัวเอง

1. **ผู้เล่นสามารถเล่นได้เพียงลำพัง** - ไม่ต้องรอคู่
2. **ยังคงมีการจับคู่ปกติ** - ระบบจับคู่ปกติก่อน
3. **ค่าธรรมเนียมยังคงหัก** - 10% จากยอดเล่น
4. **ยอดเงินคำนวณถูกต้อง** - ชนะ-แพ้ยังคงเป็นศูนย์

- ต้องมีฝั่งตรงข้าม

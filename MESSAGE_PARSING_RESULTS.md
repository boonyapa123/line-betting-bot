## 🔄 ขั้นตอนการถอดข้อความ

### ข้อความต้นฉบับ
```
"340-370 ย 100 ศ.ราช"
```

### Pattern ที่ใช้
```javascript
/^(\d+-\d+)\s+([ลย])\s+(\d+)\s+(.+)$/
```

### การแยกข้อมูล
```
340-370  →  (\d+-\d+)    ← ราคา
ย        →  ([ลย])       ← เดิมพัน
100      →  (\d+)        ← ยอดเงิน
ศ.ราช    →  (.+)        ← ชื่อบั้งไฟ
```

### ผลลัพธ์
```javascript
const [, price, side, amount, slipName] = match;

price = "340-370"
side = "ย"
amount = "100"
slipName = "ศ.ราช"
```

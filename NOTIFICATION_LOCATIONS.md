# ตำแหน่งที่ระบบส่งแจ้งเตือน

## 1. ผลการประกาศแพ้ชนะ (Result Announcement)
- **ส่งไปที่**: แชทส่วนตัวของผู้เล่น A และ B
- **เมื่อ**: ประกาศผลแพ้ชนะ (✅ ชนะ, ❌ แพ้, 🤝 เสมอ)
- **ตำแหน่ง**: `updateBetResult()` function (lines 487-554)

## 2. ยืนยันการลงทะเบียน (Registration Confirmation)
- **ส่งไปที่**: แชทส่วนตัวของผู้เล่น
- **เมื่อ**: ผู้เล่นส่งสลิปและตรวจสอบสำเร็จ
- **ตำแหน่ง**: Slip verification section (line 1100)

## 3. ข้อผิดพลาดการตรวจสอบสลิป (Slip Verification Errors)
- **ส่งไปที่**: แชทส่วนตัวของผู้เล่น
- **เมื่อ**: 
  - สลิปซ้ำ (Duplicate slip)
  - บัญชีผู้รับไม่ตรงกัน (Receiver mismatch)
  - สลิปปลอม (Fake slip)
- **ตำแหน่ง**: Slip verification error handling (lines 1659-1689)

## 4. สรุปยอดแทง (Betting Summary)
- **ส่งไปที่**: กลุ่มแชท
- **เมื่อ**: ผู้เล่นพิมพ์ "สรุปยอดแทง"
- **ตำแหน่ง**: Summary command (lines 1833-1838)

## 5. ข้อมูลบัญชีธนาคาร (Bank Account Info)
- **ส่งไปที่**: กลุ่มแชท
- **เมื่อ**: ผู้เล่นพิมพ์ "บช"
- **ตำแหน่ง**: Bank command (line 1842)

## 6. กติกาการแทง (Betting Rules)
- **ส่งไปที่**: กลุ่มแชท
- **เมื่อ**: ผู้เล่นพิมพ์ "กติกา" หรือ "กต"
- **ตำแหน่ง**: Rules command (lines 1847, 1852)

## 7. ผู้เล่นไม่ใช่เพื่อน OA (Not Friend Notification)
- **ส่งไปที่**: แชทส่วนตัวของผู้เล่น
- **เมื่อ**: ระบบไม่สามารถดึงข้อมูลโปรไฟล์ (HTTP 403/404)
- **ตำแหน่ง**: Profile fetching error (line 1972)

## 8. การจับคู่อัตโนมัติ (Auto-Matching Notification)
- **ส่งไปที่**: 
  - แชทส่วนตัวของผู้เล่น A
  - แชทส่วนตัวของผู้เล่น B
  - กลุ่มแชท
- **เมื่อ**: ระบบจับคู่เล่นอัตโนมัติสำเร็จ
- **ตำแหน่ง**: Auto-matching section (lines 2158-2165)

## 9. ผู้เล่นยังไม่ลงทะเบียน (Not Registered Warning)
- **ส่งไปที่**: 
  - แชทส่วนตัวของผู้เล่นที่ไม่ลงทะเบียน
  - **กลุ่มแชท** ✅
- **เมื่อ**: ผู้เล่นพยายามเล่นแต่ยังไม่ลงทะเบียน
- **ตำแหน่ง**: Not registered check (lines 2324-2353)

## 10. ยอดเงินไม่พอ (Insufficient Balance Warning)
- **ส่งไปที่**: 
  - แชทส่วนตัวของผู้เล่นที่เงินไม่พอ
  - **กลุ่มแชท** ✅
- **เมื่อ**: ผู้เล่นพยายามเล่นแต่เงินไม่พอ
- **ตำแหน่ง**: Insufficient balance check (lines 2374-2407)

## 11. การจับคู่เล่นสำเร็จ (Successful Pairing Notification)
- **ส่งไปที่**: 
  - แชทส่วนตัวของผู้เล่น A
  - แชทส่วนตัวของผู้เล่น B
  - กลุ่มแชท
- **เมื่อ**: ระบบจับคู่เล่นสำเร็จ (Reply Method)
- **ตำแหน่ง**: Successful pairing (lines 2446-2453)

---

## สรุป
✅ = ส่งแจ้งเตือนในกลุ่มแชท
❌ = ไม่ส่งแจ้งเตือนในกลุ่มแชท

**ปัญหา**: ข้อมูลที่ส่งไปในกลุ่มแชทอาจไม่ถูกส่งเนื่องจากปัญหา API หรือ rate limiting

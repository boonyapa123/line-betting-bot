# Balance Check Flow - ตรวจสอบเงินคงเหลือก่อนเป็นอันดับแรก

## ภาพรวม
ระบบได้ปรับปรุงให้ตรวจสอบเงินคงเหลือและชื่อผู้เล่นก่อนเป็นอันดับแรก ถ้าไม่มีเงินหรือไม่มีชื่อในระบบ ให้แจ้งเตือนทันที

## ลำดับการทำงาน

### 1. ผู้เล่นส่งข้อความเดิมพัน
```
ผู้เล่น: "ลูกชายภูน้อย ชล 150"
```

### 2. ระบบตรวจสอบตามลำดับ

#### ขั้นตอนที่ 1: ตรวจสอบคำสั่งแอดมิน
- ถ้าเป็นคำสั่งแอดมิน → ประมวลผลคำสั่ง

#### ขั้นตอนที่ 2: ตรวจสอบสถานะรอบการเดิมพัน
- ถ้ารอบปิด → แจ้งเตือน "รอบนี้ปิดการทายแล้ว"

#### ขั้นตอนที่ 3: ตรวจสอบ Reply Method
- ถ้าเป็น Reply Method → ประมวลผล Reply

#### ขั้นตอนที่ 4: Parse ข้อความเล่น (Direct Method)
- ตรวจสอบรูปแบบข้อความ
- ถ้าไม่ถูกต้อง → แจ้งเตือนข้อผิดพลาด

#### ขั้นตอนที่ 5: ตรวจสอบความถูกต้องของข้อมูล
- ตรวจสอบชื่อบั้งไฟ, ด้าน, จำนวนเงิน
- ถ้าไม่ถูกต้อง → แจ้งเตือนข้อผิดพลาด

#### ขั้นตอนที่ 6: ตรวจสอบเงินคงเหลือและชื่อผู้เล่น (อันดับแรก) ⭐
```javascript
const balanceCheck = await balanceCheckService.checkAndNotify(
  lineName,
  parsedBet.amount,
  userId,
  1, // Account 1
  groupId
);
```

**ตรวจสอบ 2 สิ่ง:**

- ค้นหาในชีท `Players`
- ถ้าไม่พบ → **แจ้งเตือนทันที** ❌

## ฟังก์ชันที่เพิ่มเติม

### 1. `isPlayerRegistered(lineName)`
ตรวจสอบว่าผู้เล่นมีชื่อในระบบหรือไม่
- ค้นหาในชีท `UsersBalance`
- ค้นหาในชีท `Players`
- Return: `true` / `false`

### 2. `notifyPlayerNotRegistered(lineName, userId, accountNumber, groupId)`
แจ้งเตือนเมื่อผู้เล่นไม่พบในระบบ (ยังไม่ลงทะเบียน)
- ส่งข้อความส่วนตัว
- ส่งข้อความในกลุ่ม (ถ้ามี groupId)
- ข้อความชัดเจนว่าต้องเติมเงินเพื่อลงทะเบียน

### 3. `checkBalance(lineName, requiredAmount)`
ตรวจสอบเงินคงเหลือ (ตรวจสอบชื่อผู้เล่นก่อน)
- Return: `{sufficient, currentBalance, shortfall, registered, message}`

### 4. `checkAndNotify(lineName, requiredAmount, userId, accountNumber, groupId)`
ตรวจสอบและแจ้งเตือน (ตรวจสอบชื่อผู้เล่นก่อน)
- ถ้าผู้เล่นไม่พบ → แจ้งเตือนทันที
- ถ้ายอดเงินไม่พอ → แจ้งเตือนทันที
- Return: `{sufficient, currentBalance, shortfall, registered, message}`

## ข้อดีของการปรับปรุง

✅ **ตรวจสอบชื่อผู้เล่นก่อนเป็นอันดับแรก**
- ไม่ต้องรอการตรวจสอบเงินถ้าผู้เล่นไม่มีชื่อในระบบ

✅ **แจ้งเตือนทันที**
- ผู้เล่นรู้ทันทีว่าเป็นปัญหาอะไร
- ไม่มีการบันทึกการเล่นที่ไม่ถูกต้อง

✅ **ข้อความชัดเจนว่าต้องเติมเงิน**
- ผู้เล่นที่ยังไม่ลงทะเบียน → "เติมเงินเพื่อลงทะเบียน"
- ผู้เล่นที่ลงทะเบียนแล้วแต่เงินไม่พอ → "เติมเงิน"

✅ **ส่งข้อความ 2 ช่องทาง**
- ข้อความส่วนตัว (ผู้เล่นรู้ทันที)
- ข้อความในกลุ่ม (สมาชิกอื่นรู้ว่าเกิดอะไรขึ้น)

✅ **ลดข้อมูลขยะในระบบ**
- ไม่บันทึกการเล่นที่ไม่ถูกต้อง
- ระบบสะอาดและเรียบร้อย

## 🎯 Flow ตรวจสอบผลลัพธ์ (checkPriceRangeResult)

```
┌─────────────────────────────────────────────────────────────────┐
│ Input: bet1, bet2, score                                        │
│ เช่น: A(300-340 ล), B(reply ล), score=320                      │
└────────────────────┬────────────────────────────────────────────┘
                     │
                     ▼
        ┌────────────────────────────────┐
        │ ตรวจสอบช่วงราคา               │
        │ hasPriceRange1 = true          │
        │ hasPriceRange2 = false         │
        └────────────────┬───────────────┘
                         │
                         ▼
        ┌────────────────────────────────┐
        │ เฉพาะ bet1 มีช่วงราคา         │
        │ (bet2 เป็น reply)              │
        └────────────────┬───────────────┘
                         │
                         ▼
        ┌────────────────────────────────┐
        │ ตรวจสอบ score ในช่วง           │
        │ 320 >= 300 && 320 <= 340?      │
        └────────────┬───────────────────┘
                     │
        ┌────────────┴────────────┐
        │                         │
        ▼                         ▼
    ┌─────────┐            ┌──────────┐
    │ YES     │            │ NO       │
    │ (ใน)    │            │ (นอก)    │
    └────┬────┘            └────┬─────┘
         │                      │
         ▼                      ▼
    ┌─────────┐        ┌──────────────────┐
    │ isDraw  │        │ ตรวจสอบ ย/ล      │
    │ = true  │        │ isYang1 = false  │
    │ ⛔️      │        │ isLow1 = true    │
    └─────────┘        │ score > max?     │
                       │ 320 > 340?       │
                       └────┬─────────────┘
                            │
                    ┌───────┴────────┐
                    │                │
                    ▼                ▼
                ┌────────┐      ┌────────┐
                │ NO     │      │ YES    │
                │ (ไม่)  │      │ (ใช่)  │
                └────┬───┘      └────┬───┘
                     │               │
                     ▼               ▼
                ┌────────┐      ┌────────┐
                │ return │      │ return │
                │ null   │      │ winner │
                │        │      │ = bet1 │
                └────────┘      └────────┘
```

## 📍 ตัวอย่าง: ผลออกในช่วง (เสมอ)

Input:
┌─────────────────────────────────────────────────────────────────┐
│ Bet A: paa"BOY" เล่น 300-340 ล 30 ฟ้า                          │

│ Result: 320                                                     │
└─────────────────────────────────────────────────────────────────┘

│ {                                                               │
│   isDraw: true,                                                 │

## 🔑 Key Points

1. **ตรวจสอบช่วงราคา** - ใช้ `includes('-')` เพื่อตรวจสอบว่ามีช่วงราคา
2. **ตรวจสอบ ย/ล** - ใช้ regex `/\s+ย\s+/` และ `/\s+ล\s+/`
3. **บันทึกผล** - อัปเดต 4 คอลัมน์: I, J, S, T
4. **ค่าธรรมเนียม** - 10% สำหรับชนะ-แพ้, 5% สำหรับเสมอ
5. **ยอดเดิมพัน** - ใช้ยอดที่น้อยกว่า (ยอดที่จับคู่ได้จริง)

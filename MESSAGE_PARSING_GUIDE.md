# 📝 วิธีการถอดข้อความ (Message Parsing)

## 📋 ภาพรวม

ระบบถอดข้อความโดยใช้ Regex Pattern เพื่อแยกข้อมูลออกจากข้อความ

---

## 🔍 ตัวอย่างข้อความ

```
"320-340 ย 100 คำไผ่"
```

---

## 🔄 ขั้นตอนการถอดข้อความ

### 1️⃣ ตรวจสอบรูปแบบ

ระบบจะตรวจสอบรูปแบบข้อความตามลำดับ:

1. **วิธีที่ 1 (ราคาช่าง) - รูปแบบเดิม**
   ```
   Pattern: /^(.+?)\s+(ชล\.|ชถ\.)\s+(\d+)$/
   ตัวอย่าง: "ฟ้าหลังฝน ชล. 500"
   ```

2. **วิธีที่ 1 (ราคาช่าง) - รูปแบบใหม่**
   ```
   Pattern: /^(ชล|ชถ|ชย)\s+(\d+)\s+(.+)$/
   ตัวอย่าง: "ชล 500 ฟ้าหลังฝน"
   ```

3. **วิธีที่ 2 (ราคาคะแนน) - รูปแบบเดิม**
   ```
   Pattern: /^(.+?)\s+([ลย]\.)\s+(\d+)\s+(.+)$/
   ตัวอย่าง: "0/3(300-330) ล. 500 ฟ้าหลังฝน"
   ```

4. **วิธีที่ 2 (ราคาคะแนน) - รูปแบบใหม่**
   ```
   Pattern: /^(.+?)\s+([ลย])\s+(\d+)\s+(.+)$/
   ตัวอย่าง: "0/3(300-330) ล 500 ฟ้าหลังฝน"
   ```

5. **วิธีที่ 2 (ราคาคะแนน) - รูปแบบง่าย** ← ตรงกับตัวอย่าง
   ```
   Pattern: /^(\d+-\d+)\s+([ลย])\s+(\d+)\s+(.+)$/
   ตัวอย่าง: "320-340 ย 100 คำไผ่"
   ```

## 📊 ตัวอย่างการถอดข้อความ

### ข้อความ: "320-340 ย 100 คำไผ่"

**ขั้นตอน 1: ตรวจสอบรูปแบบ**

```javascript
const message = "320-340 ย 100 คำไผ่";
const pattern = /^(\d+-\d+)\s+([ลย])\s+(\d+)\s+(.+)$/;
const match = message.match(pattern);
```

**ผลลัพธ์:**
```javascript
match = [
  "320-340 ย 100 คำไผ่",  // [0] ข้อความทั้งหมด
  "320-340",              // [1] ราคา
  "ย",                    // [2] ฝั่ง
  "100",                  // [3] จำนวนเงิน
  "คำไผ่"                 // [4] ชื่อบั้งไฟ
]
```

**ขั้นตอน 2: แยกข้อมูล**

```javascript
const [, price, side, amount, slipName] = match;

// price = "320-340"
// side = "ย"
// amount = "100"
// slipName = "คำไผ่"
```

**ขั้นตอน 3: แปลงข้อมูล**

```javascript
const sideMap = {
  'ล': 'ไล่',
  'ย': 'ยั้ง',
};

const parsedBet = {
  success: true,
  method: 2,
  price: "320-340",
  side: sideMap['ย'] = "ยั้ง",
  sideCode: "ย",
  amount: parseInt("100") = 100,
  slipName: "คำไผ่",
  timestamp: "2026-03-04T..."
};
```

## 📌 ข้อมูลที่ถอดออกมา

| ข้อมูล | ตัวอย่าง | หมายเหตุ |
|--------|---------|---------|
| `price` | "320-340" | ราคาคะแนน (วิธีที่ 2 เท่านั้น) |
| `side` | "ยั้ง" | ฝั่งเต็ม (ไล่/ยั้ง/ถอย) |
| `sideCode` | "ย" | ฝั่งย่อ (ล/ย/ชล/ชถ) |
| `amount` | 100 | จำนวนเงิน (ตัวเลข) |
| `slipName` | "คำไผ่" | ชื่อบั้งไฟ |
| `method` | 2 | วิธีการเล่น (1 หรือ 2) |
| `timestamp` | "2026-03-04T..." | เวลาที่ส่งข้อความ |

## 🚀 ขั้นตอนการใช้งาน

```javascript
// 1. Import Service
const BettingMessageParserService = require('./bettingMessageParserService');

// 2. Parse ข้อความ
const message = "320-340 ย 100 คำไผ่";
const parsedBet = BettingMessageParserService.parseMessage(message);

// 3. ตรวจสอบผลลัพธ์
if (parsedBet.success) {
  console.log('ราคา:', parsedBet.price);
  console.log('ฝั่ง:', parsedBet.side);
  console.log('จำนวนเงิน:', parsedBet.amount);
  console.log('ชื่อบั้งไฟ:', parsedBet.slipName);
} else {
  console.log('ข้อผิดพลาด:', parsedBet.error);
  console.log('คำแนะนำ:', parsedBet.hint);
}
```

## 📝 หมายเหตุ

- ระบบตรวจสอบรูปแบบตามลำดับ (ตรวจสอบรูปแบบแรกก่อน)
- ถ้าข้อความตรงรูปแบบใด ระบบจะส่งผลลัพธ์ทันที
- ถ้าไม่ตรงรูปแบบใด ระบบจะส่งข้อความแจ้งข้อผิดพลาด
- ข้อความต้องมีเว้นวรรค (space) ระหว่างข้อมูล
- ระบบรองรับทั้งรูปแบบเดิมและรูปแบบใหม่

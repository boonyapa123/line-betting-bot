## 📌 หมายเหตุ

- **Column A-G**: ข้อมูล User A (บันทึกเมื่อ User A ส่งข้อความ)
- **Column H-M**: ข้อมูล User B (บันทึกเมื่อจับคู่สำเร็จ)
- **Column I-J, S-T**: ผลลัพธ์ (อัปเดตเมื่อประกาศผล)
- **Column U**: ทำเครื่องหมายจับคู่อัตโนมัติ (AUTO)
- **Column N, O, P, R**: ยังไม่ใช้ (สำรองไว้สำหรับอนาคต)

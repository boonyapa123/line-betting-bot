# ⚡ การประกาศผล - ข้อมูลอ้างอิงด่วน

## 📝 คำสั่งประกาศผล

```
:สรุป [ชื่อบั้งไฟ] [คะแนน]
```

### ตัวอย่าง
```
:สรุป ฟ้า 370
:สรุป แอดเทวดา 325
:สรุป เป็ด 450
```

---

## 🔍 ตรรกะการตรวจสอบผล

### ผลออกในช่วง → ⛔️ เสมอ
```
User A: ล/325-340/100ฟ้า
User B: reply ล
ผลออก: 330 (อยู่ในช่วง 325-340)
→ ⛔️ เสมอ (ทั้งสองเสีย 5%)
```

### ผลออกต่ำกว่าช่วง + User A เป็น ย → ✅ A ชนะ
```
User A: ย/310-320/20ฟ้า
User B: reply ต
ผลออก: 305 (ต่ำกว่า 310)
→ ✅ A ชนะ (A ได้ 90%, B เสีย 100%)
```

### ผลออกสูงกว่าช่วง + User A เป็น ล → ✅ A ชนะ
```
User A: ล/300-340/30ฟ้า
User B: reply ล
ผลออก: 350 (สูงกว่า 340)
→ ✅ A ชนะ (A ได้ 90%, B เสีย 100%)
```

---

## 💰 ค่าธรรมเนียม

| สถานการณ์ | ค่าธรรมเนียม | ตัวอย่าง (100 บาท) |
|---------|-----------|-----------------|
| ชนะ-แพ้ | 10% | ผู้ชนะ: +90, ผู้แพ้: -100 |
| เสมอ | 5% | ทั้งสอง: -5 |

---

## 📊 ผลลัพธ์ที่บันทึก

| Column | ชื่อ | ค่า | ตัวอย่าง |
|--------|------|-----|---------|
| I | ผลที่ออก | ตัวเลข | 370 |
| J | ผลแพ้ชนะ A | ✅/❌/⛔️ | ✅ |
| K | ผลแพ้ชนะ B | ✅/❌/⛔️ | ❌ |
| S | ยอดเงิน A | +/- | +90 |
| T | ยอดเงิน B | +/- | -100 |

---

## ⚠️ ข้อผิดพลาดที่พบบ่อย

| ข้อผิดพลาด | ตัวอย่าง | วิธีแก้ |
|---------|---------|--------|
| ลืมคำสั่ง | `สรุป ฟ้า 370` | ต้องใช้ `:สรุป` |
| ชื่อบั้งไฟไม่ตรง | `:สรุป ฟ้า 370` (แต่ผู้เล่นใช้ "ฟ้าหลังฝน") | ใช้ชื่อเดียวกัน |
| คะแนนไม่ใช่ตัวเลข | `:สรุป ฟ้า สามร้อยเจ็ด` | ใช้ตัวเลข `370` |

---

## 🎯 ความหมายของ ย/ล

- **ย** = ต่ำ → ชนะเมื่อผลต่ำกว่าช่วง
- **ล** = สูง → ชนะเมื่อผลสูงกว่าช่วง

---

## 📱 ตัวอย่างการใช้งาน

### ประกาศผลฟ้า 370

```
Admin: :สรุป ฟ้า 370

ระบบค้นหา: การเล่นบั้งไฟ "ฟ้า" ทั้งหมด

ตรวจสอบคู่ที่ 1:
- User A: ไล่/325-340/100ฟ้า
- User B: reply ล
- ผลออก: 370 > 340 (สูงกว่าช่วง)
- User A เป็น ล (สูง) → ✅ A ชนะ
- บันทึก: I=370, J=✅, K=❌, S=+90, T=-100

ตรวจสอบคู่ที่ 2:
- User A: ย/360-380/50ฟ้า
- User B: reply ย
- ผลออก: 370 (อยู่ในช่วง)
- ⛔️ เสมอ
- บันทึก: I=370, J=⛔️, K=⛔️, S=-2.5, T=-2.5

ตรวจสอบคู่ที่ 3:
- User A: ล/300-350/200ฟ้า
- User B: reply ล
- ผลออก: 370 > 350 (สูงกว่าช่วง)
- User A เป็น ล (สูง) → ✅ A ชนะ
- บันทึก: I=370, J=✅, K=❌, S=+180, T=-200

แจ้งเตือน: ส่วนตัว + กลุ่ม
```

## 🔄 ขั้นตอนการประกาศผล

```
1. ประกาศผล (:สรุป ฟ้า 370)
   ↓
2. ค้นหาการเล่น
   ↓
3. ตรวจสอบผล
   ↓
4. คำนวณค่าธรรมเนียม
   ↓
5. บันทึกผล
   ↓
6. อัปเดตยอดเงิน
   ↓
7. แจ้งเตือน LINE
   ↓
8. ล้างข้อมูล
   ↓
9. ปิดรอบ
```

## 📞 ติดต่อ

หากมีปัญหา:

2. ตรวจสอบคะแนน
3. ตรวจสอบคำสั่ง `:สรุป`

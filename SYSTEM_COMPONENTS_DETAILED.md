# ส่วนประกอบของระบบ - รายละเอียด

## 🔧 ส่วนประกอบหลัก

### 1. Slip Verification (Account 3)
**ไฟล์:** `services/betting/slip2GoImageVerificationService.js`

**หน้าที่:**
- รับรูปสลิปจากผู้เล่น
- ตรวจสอบผ่าน Slip2Go API
- ตรวจสอบชื่อผู้รับเงิน, จำนวนเงิน, วันที่

**ผลลัพธ์:**
```
✅ สลิปถูกต้อง
  → อัปเดต Players Sheet
  → บันทึก Transactions Sheet
  → ส่งแจ้งเตือน LINE ส่วนตัว

❌ สลิปไม่ถูกต้อง
  → ส่งแจ้งเตือน: "สลิปไม่ถูกต้อง"
```

---

### 2. Balance Check Service
**ไฟล์:** `services/betting/balanceCheckService.js`

**หน้าที่:**
- ตรวจสอบยอดเงินของผู้เล่น
- ค้นหาผู้เล่นจาก LINE name, Linked IDs, User ID
- ส่งแจ้งเตือนเมื่อยอดเงินไม่พอ

**ฟังก์ชันหลัก:**
```javascript
checkBalance(lineName, requiredAmount)
  → { sufficient: true/false, currentBalance, shortfall }

getUserBalance(lineName)
  → ยอดเงินปัจจุบัน

checkAndNotify(lineName, requiredAmount, userId)
  → ตรวจสอบและส่งแจ้งเตือน
```

---

### 3. Betting Message Parser
**ไฟล์:** `services/betting/bettingMessageParserService.js`

**หน้าที่:**
- แยกข้อมูลจากข้อความแทง
- ดึง: ชื่อบั้งไฟ, ประเภทเล่น, ยอดเงิน

**ตัวอย่าง:**
```
Input: "ลูกชายภูน้อย ชล 150"
Output: {
  fireworkName: "ลูกชายภูน้อย",
  betType: "ชล",
  betAmount: 150
}
```

---

### 4. Betting Pairing Service
**ไฟล์:** `services/betting/bettingPairingService.js`

**หน้าที่:**
- จับคู่ผู้เล่น 2 คน
- ตรวจสอบชื่อบั้งไฟและประเภทเล่น
- บันทึกการเดิมพันลง Bets Sheet

**ฟังก์ชันหลัก:**
```javascript
recordBet(pair, userAName, userBName, groupName, method)
  → บันทึกลง Bets Sheet
  → Columns: A-N (14 columns)
```

---

### 5. Result Settlement Service
**ไฟล์:** `services/betting/resultSettlementService.js`

**หน้าที่:**
- อัปเดตผลลัพธ์ใน Bets Sheet
- คำนวณยอดเงินใหม่
- บันทึก Transactions Sheet

**ตัวอย่าง:**
```
User A ชนะ:
  ยอดเงินใหม่ = ยอดเงินเดิม + ยอดเงินเดิมพัน

User B แพ้:
  ยอดเงินใหม่ = ยอดเงินเดิม - ยอดเงินเดิมพัน
```

---

### 6. LINE Notification Service
**ไฟล์:** `services/line/lineNotificationService.js`

**หน้าที่:**
- ส่งข้อความไปยัง LINE ส่วนตัว
- ส่งข้อความไปยัง กลุ่มแชท
- ใช้ Account 1, 2, หรือ 3

**ฟังก์ชันหลัก:**
```javascript
sendPrivateMessage(userId, message)
  → ส่งไปยัง LINE ส่วนตัว

sendGroupMessage(groupId, message)
  → ส่งไปยัง กลุ่มแชท
```

## 📊 ข้อมูลไหลผ่านระบบ

### 1. เติมเงิน
```
สลิป (รูป)
  ↓
Slip2Go API (ตรวจสอบ)
  ↓
Players Sheet (อัปเดตยอดเงิน)
Transactions Sheet (บันทึก)
UsersBalance Sheet (อัปเดต)
  ↓
LINE ส่วนตัว (แจ้งเตือน)
```

### 2. เช็คยอดเงิน
```
ข้อความแทง
  ↓
ดึงชื่อ LINE
  ↓
ค้นหาใน Players Sheet
  ↓
ตรวจสอบยอดเงิน
  ↓
LINE ส่วนตัว (แจ้งเตือน)
```

### 3. จับคู่เล่น
```
ข้อความแทง User A
  ↓
ข้อความแทง User B (ตอบกลับ)
  ↓
ตรวจสอบชื่อบั้งไฟ
ตรวจสอบประเภทเล่น
ตรวจสอบยอดเงิน
  ↓
Bets Sheet (บันทึก)
  ↓
LINE ส่วนตัว + กลุ่ม (แจ้งเตือน)
```

### 4. ประกาศผล
```
ข้อความประกาศผล
  ↓
ค้นหาการเดิมพันใน Bets Sheet
  ↓
อัปเดตผลลัพธ์
  ↓
Bets Sheet (อัปเดต)
Players Sheet (อัปเดตยอดเงิน)
Transactions Sheet (บันทึก)
UsersBalance Sheet (อัปเดต)
  ↓
LINE ส่วนตัว + กลุ่ม (แจ้งเตือน)
```

## 📈 ตัวอย่างการทำงาน

### ตัวอย่าง 1: เติมเงิน
```
1. ผู้เล่น: ส่งรูปสลิป 500 บาท
2. ระบบ: ตรวจสอบสลิป
3. ระบบ: ✅ สลิปถูกต้อง
4. ระบบ: อัปเดต Players Sheet
   - ยอดเงินเดิม: 5000 บาท
   - ยอดเงินใหม่: 5500 บาท
5. ระบบ: บันทึก Transactions Sheet
   - ประเภท: เติมเงิน
   - จำนวน: +500 บาท
6. ระบบ: ส่งแจ้งเตือน LINE ส่วนตัว
   "✅ ยืนยันการเติมเงิน 500 บาท สำเร็จ
    ยอดเงินปัจจุบัน: 5500 บาท"
```

### ตัวอย่าง 2: เช็คยอดเงิน
```
1. ผู้เล่น: "ลูกชายภูน้อย ชล 150"
2. ระบบ: ดึงชื่อ LINE = "💓Noon💓"
3. ระบบ: ค้นหาใน Players Sheet
4. ระบบ: ✅ พบผู้เล่น
5. ระบบ: ตรวจสอบยอดเงิน
   - ยอดเงินปัจจุบัน: 5500 บาท
   - ต้องการ: 150 บาท
6. ระบบ: ✅ ยอดเงินพอ
7. ระบบ: เก็บข้อความไว้ในหน่วยความจำ
   รอผู้เล่นคนที่ 2 ตอบกลับ
```

### ตัวอย่าง 3: จับคู่เล่น
```
1. ผู้เล่น A: "ลูกชายภูน้อย ชล 150"
2. ผู้เล่น B: ตอบกลับด้วย "ต"
3. ระบบ: ตรวจสอบชื่อบั้งไฟ
   - User A: "ลูกชายภูน้อย"
   - User B: "ลูกชายภูน้อย"
   - ✅ ตรงกัน
4. ระบบ: ตรวจสอบประเภทเล่น
   - User A: "ชล"
   - User B: "ต"
   - ✅ ตรงข้าม
5. ระบบ: ตรวจสอบยอดเงิน
   - User A: 5500 บาท ✅
   - User B: 4850 บาท ✅
6. ระบบ: บันทึก Bets Sheet
7. ระบบ: ส่งแจ้งเตือนกลุ่ม
   "✅ ทำการเดิมพันสำเร็จ
    💓Noon💓 เล่น ชล 150 บาท
    Pam Yuthida เล่น ต 150 บาท"
```

### ตัวอย่าง 4: ประกาศผล
```
1. ผู้ดำเนินการ: "ลูกชายภูน้อย ชล"
2. ระบบ: ค้นหาการเดิมพัน
   - ชื่อบั้งไฟ: "ลูกชายภูน้อย"
   - ยังไม่มีผลลัพธ์
3. ระบบ: ✅ พบการเดิมพัน
4. ระบบ: อัปเดตผลลัพธ์
   - User A (ชล): ชนะ
   - User B (ต): แพ้
5. ระบบ: อัปเดตยอดเงิน
   - User A: 5500 + 150 = 5650 บาท
   - User B: 4850 - 150 = 4700 บาท
6. ระบบ: บันทึก Transactions Sheet
7. ระบบ: ส่งแจ้งเตือน LINE ส่วนตัว
   User A: "🎉 ยินดีด้วย! คุณชนะ 150 บาท
            ยอดเงินปัจจุบัน: 5650 บาท"
   User B: "😢 ขออภัย คุณแพ้ 150 บาท
            ยอดเงินปัจจุบัน: 4700 บาท"
8. ระบบ: ส่งแจ้งเตือนกลุ่ม
   "✅ ประกาศผลแล้ว
    🎆 ลูกชายภูน้อย: ชล
    🏆 ผู้ชนะ: 💓Noon💓 (150 บาท)
    😢 ผู้แพ้: Pam Yuthida (150 บาท)"
```

# การทำงานของ Google Sheets - รายละเอียด

## 📊 ภาพรวม 3 Sheet

```
┌─────────────────────────────────────────────────────────────┐
│                    PLAYERS SHEET (หลัก)                     │
│  - ข้อมูลผู้เล่น (ชื่อ, User ID, Linked IDs)               │
│  - ยอดเงินปัจจุบัน (อัปเดตจาก Transactions)                │
└─────────────────────────────────────────────────────────────┘
                              ↓
                    (อัปเดตจาก Transactions)
                              ↓
┌─────────────────────────────────────────────────────────────┐
│              TRANSACTIONS SHEET (บันทึก)                    │
│  - บันทึกทุกการเปลี่ยนแปลงยอดเงิน                          │
│  - เติมเงิน, ชนะ, แพ้                                      │
│  - ใช้สำหรับ Audit Trail                                   │
└─────────────────────────────────────────────────────────────┘
                              ↓
                    (สำเนาจาก Players)
                              ↓
┌─────────────────────────────────────────────────────────────┐
│            USERSBALANCE SHEET (ยอดเงินปัจจุบัน)            │
│  - สำเนาของยอดเงินจาก Players Sheet                        │
│  - ใช้สำหรับการค้นหาเร็ว                                   │
│  - อัปเดตอัตโนมัติจาก Players Sheet                        │
└─────────────────────────────────────────────────────────────┘
```

---

## 1️⃣ PLAYERS SHEET (ชีทหลัก)

### โครงสร้าง
```
| A | B | C | D | E | F | G | H | I | J | K |
|---|---|---|---|---|---|---|---|---|---|---|
| User ID | ชื่อ LINE | Linked IDs | ชื่อเล่น | ยอดเงิน | ... |
```

### รายละเอียดคอลัมน์

| Column | ชื่อ | ประเภท | ตัวอย่าง | หมายเหตุ |
|--------|------|--------|---------|---------|
| A | User ID | Text | U2fbb940e366149fc8dbdf92b92a81b60 | ไม่ซ้ำกัน |
| B | ชื่อ LINE | Text | 💓Noon💓 | ใช้ค้นหาหลัก |
| C | Linked IDs | JSON Array | ["1000", "2000"] | ลิงก์ ID อื่น |
| D | ชื่อเล่น | Text | Noon | ชื่อเล่นในเกม |
| E | ยอดเงิน | Number | 5500 | อัปเดตจาก Transactions |

### ตัวอย่างข้อมูล
```
| User ID | ชื่อ LINE | Linked IDs | ชื่อเล่น | ยอดเงิน |
|---------|-----------|-----------|---------|---------|
| U2fbb940e366149fc8dbdf92b92a81b60 | 💓Noon💓 | ["1000"] | Noon | 5500 |
| Ubb4a196cd0dc0c3ca6211a9bafecb513 | Pam Yuthida | [] | Pam | 4700 |
| U123456789 | สมชาย | ["3000", "4000"] | สมชาย | 3200 |
```

### การอัปเดต

**เมื่อเติมเงิน:**
```
1. ตรวจสอบสลิป ✅
2. ค้นหาผู้เล่นใน Players Sheet
3. อัปเดตคอลัมน์ E (ยอดเงิน)
   ยอดเงินใหม่ = ยอดเงินเดิม + จำนวนเติม
4. บันทึก Transactions Sheet
5. อัปเดต UsersBalance Sheet
```

**เมื่อประกาศผล:**
```
1. ค้นหาการเดิมพันใน Bets Sheet
2. ค้นหาผู้เล่นใน Players Sheet
3. อัปเดตคอลัมน์ E (ยอดเงิน)
   - ผู้ชนะ: ยอดเงินใหม่ = ยอดเงินเดิม + ยอดเงินเดิมพัน
   - ผู้แพ้: ยอดเงินใหม่ = ยอดเงินเดิม - ยอดเงินเดิมพัน
4. บันทึก Transactions Sheet
5. อัปเดต UsersBalance Sheet
```

## 2️⃣ TRANSACTIONS SHEET (บันทึกการเปลี่ยนแปลง)

### โครงสร้าง
```
| A | B | C | D | E | F | G | H |
|---|---|---|---|---|---|---|---|
| Timestamp | ชื่อผู้เล่น | ประเภท | จำนวน | ยอดเงินใหม่ | หมายเหตุ | ... |
```

### รายละเอียดคอลัมน์

| Column | ชื่อ | ประเภท | ตัวอย่าง | หมายเหตุ |
|--------|------|--------|---------|---------|
| A | Timestamp | DateTime | 2026-03-02T15:25:30 | เวลาที่เกิดการเปลี่ยนแปลง |
| B | ชื่อผู้เล่น | Text | 💓Noon💓 | ชื่อ LINE |
| C | ประเภท | Text | เติมเงิน / ชนะเดิมพัน / แพ้เดิมพัน | ประเภทการเปลี่ยนแปลง |
| D | จำนวน | Number | +500 / +150 / -150 | บวก = เพิ่ม, ลบ = ลด |
| E | ยอดเงินใหม่ | Number | 5500 | ยอดเงินหลังการเปลี่ยนแปลง |
| F | หมายเหตุ | Text | สลิป #12345 / ลูกชายภูน้อย ชล | รายละเอียดเพิ่มเติม |

### ตัวอย่างข้อมูล
```
| Timestamp | ชื่อผู้เล่น | ประเภท | จำนวน | ยอดเงินใหม่ | หมายเหตุ |
|-----------|-----------|--------|--------|------------|---------|
| 2026-03-02T15:00:00 | 💓Noon💓 | เติมเงิน | +500 | 5500 | สลิป #12345 |
| 2026-03-02T15:25:30 | 💓Noon💓 | ชนะเดิมพัน | +150 | 5650 | ลูกชายภูน้อย ชล |
| 2026-03-02T15:25:30 | Pam Yuthida | แพ้เดิมพัน | -150 | 4700 | ลูกชายภูน้อย ต |
```

### ประเภทการเปลี่ยนแปลง

| ประเภท | จำนวน | ตัวอย่าง | หมายเหตุ |
|--------|--------|---------|---------|
| เติมเงิน | + | +500 | ผู้เล่นส่งสลิป |
| ชนะเดิมพัน | + | +150 | ผู้เล่นชนะการเดิมพัน |
| แพ้เดิมพัน | - | -150 | ผู้เล่นแพ้การเดิมพัน |
| ปรับยอดเงิน | +/- | +100 / -50 | แอดมินปรับยอดเงิน |

### การบันทึก

**เมื่อเติมเงิน:**
```
Timestamp: 2026-03-02T15:00:00
ชื่อผู้เล่น: 💓Noon💓
ประเภท: เติมเงิน
จำนวน: +500
ยอดเงินใหม่: 5500
หมายเหตุ: สลิป #12345
```

**เมื่อประกาศผล (ผู้ชนะ):**
```
Timestamp: 2026-03-02T15:25:30
ชื่อผู้เล่น: 💓Noon💓
ประเภท: ชนะเดิมพัน
จำนวน: +150
ยอดเงินใหม่: 5650
หมายเหตุ: ลูกชายภูน้อย ชล
```

**เมื่อประกาศผล (ผู้แพ้):**
```
Timestamp: 2026-03-02T15:25:30
ชื่อผู้เล่น: Pam Yuthida
ประเภท: แพ้เดิมพัน
จำนวน: -150
ยอดเงินใหม่: 4700
หมายเหตุ: ลูกชายภูน้อย ต
```

## 3️⃣ USERSBALANCE SHEET (ยอดเงินปัจจุบัน)

### โครงสร้าง
```
| A | B | C |
|---|---|---|
| User ID | ชื่อ LINE | ยอดเงิน |
```

### รายละเอียดคอลัมน์

| Column | ชื่อ | ประเภท | ตัวอย่าง | หมายเหตุ |
|--------|------|--------|---------|---------|
| A | User ID | Text | U2fbb940e366149fc8dbdf92b92a81b60 | ไม่ซ้ำกัน |
| B | ชื่อ LINE | Text | 💓Noon💓 | ชื่อ LINE |
| C | ยอดเงิน | Number | 5650 | สำเนาจาก Players Sheet |

### ตัวอย่างข้อมูล
```
| User ID | ชื่อ LINE | ยอดเงิน |
|---------|-----------|---------|
| U2fbb940e366149fc8dbdf92b92a81b60 | 💓Noon💓 | 5650 |
| Ubb4a196cd0dc0c3ca6211a9bafecb513 | Pam Yuthida | 4700 |
| U123456789 | สมชาย | 3200 |
```

### การอัปเดต

**อัตโนมัติจาก Players Sheet:**
```
1. ระบบตรวจสอบ Players Sheet
2. ถ้า ยอดเงิน ใน Players Sheet เปลี่ยน
3. อัปเดต UsersBalance Sheet ให้ตรงกัน
```

### ใช้งาน

**ค้นหาเร็ว:**
```
เมื่อต้องการตรวจสอบยอดเงินของผู้เล่น
- ค้นหาใน UsersBalance Sheet (เร็วกว่า)
- แทนการค้นหาใน Players Sheet (ช้ากว่า)
```

## 4️⃣ BETS SHEET (บันทึกการเดิมพัน)

### โครงสร้าง
```
| A | B | C | D | E | F | G | H | I | J | K | L | M | N |
|---|---|---|---|---|---|---|---|---|---|---|---|---|---|
| Timestamp | User A ID | ชื่อ A | ข้อความ A | ชื่อบั้งไฟ | รายการเล่น A | ยอดเงิน | User B ID | ชื่อ B | ข้อความ B | รายการเล่น B | ผลลัพธ์ A | ผลลัพธ์ B | หมายเหตุ |
```

### รายละเอียดคอลัมน์

| Column | ชื่อ | ประเภท | ตัวอย่าง | หมายเหตุ |
|--------|------|--------|---------|---------|
| A | Timestamp | DateTime | 2026-03-02T15:25:30 | เวลาที่บันทึก |
| B | User A ID | Text | U2fbb940e366149fc8dbdf92b92a81b60 | User ID ของผู้เล่น A |
| C | ชื่อ A | Text | 💓Noon💓 | ชื่อ LINE ของผู้เล่น A |
| D | ข้อความ A | Text | ลูกชายภูน้อย ชล 150 | ข้อความแทงเต็ม |
| E | ชื่อบั้งไฟ | Text | ลูกชายภูน้อย | ชื่อบั้งไฟ |
| F | รายการเล่น A | Text | ชล | ประเภทเล่นของ User A |
| G | ยอดเงิน | Number | 150 | ยอดเงินเดิมพัน |
| H | User B ID | Text | Ubb4a196cd0dc0c3ca6211a9bafecb513 | User ID ของผู้เล่น B |
| I | ชื่อ B | Text | Pam Yuthida | ชื่อ LINE ของผู้เล่น B |
| J | ข้อความ B | Text | ต | ข้อความแทงของ User B |
| K | รายการเล่น B | Text | ต | ประเภทเล่นของ User B |
| L | ผลลัพธ์ A | Text | ชล / ต / - | ผลลัพธ์ของ User A |
| M | ผลลัพธ์ B | Text | ต / ชล / - | ผลลัพธ์ของ User B |
| N | หมายเหตุ | Text | Reply Method | วิธีการจับคู่ |

### ตัวอย่างข้อมูล
```
| Timestamp | User A ID | ชื่อ A | ข้อความ A | ชื่อบั้งไฟ | รายการเล่น A | ยอดเงิน | User B ID | ชื่อ B | ข้อความ B | รายการเล่น B | ผลลัพธ์ A | ผลลัพธ์ B | หมายเหตุ |
|-----------|-----------|--------|-----------|-----------|-------------|---------|-----------|--------|-----------|-------------|-----------|-----------|---------|
| 2026-03-02T15:25:30 | U2fbb940e366149fc8dbdf92b92a81b60 | 💓Noon💓 | ลูกชายภูน้อย ชล 150 | ลูกชายภูน้อย | ชล | 150 | Ubb4a196cd0dc0c3ca6211a9bafecb513 | Pam Yuthida | ต | ต | ชล | ต | Reply Method |
```

### การบันทึก

**เมื่อจับคู่เล่นสำเร็จ:**
```
Timestamp: 2026-03-02T15:25:30
User A ID: U2fbb940e366149fc8dbdf92b92a81b60
ชื่อ A: 💓Noon💓
ข้อความ A: ลูกชายภูน้อย ชล 150
ชื่อบั้งไฟ: ลูกชายภูน้อย
รายการเล่น A: ชล
ยอดเงิน: 150
User B ID: Ubb4a196cd0dc0c3ca6211a9bafecb513
ชื่อ B: Pam Yuthida
ข้อความ B: ต
รายการเล่น B: ต
ผลลัพธ์ A: (ว่างไว้)
ผลลัพธ์ B: (ว่างไว้)
หมายเหตุ: Reply Method
```

### การอัปเดต

**เมื่อประกาศผล:**
```
ผลลัพธ์ A: ชล (ชนะ)
ผลลัพธ์ B: ต (แพ้)
```

## 🔄 ความสัมพันธ์ระหว่าง Sheet

### 1. เติมเงิน
```
Players Sheet (อัปเดตยอดเงิน)
    ↓
Transactions Sheet (บันทึก)
    ↓
UsersBalance Sheet (อัปเดต)
```

### 2. จับคู่เล่น
```
Players Sheet (ตรวจสอบยอดเงิน)
    ↓
Bets Sheet (บันทึก)
```

### 3. ประกาศผล
```
Bets Sheet (อัปเดตผลลัพธ์)
    ↓
Players Sheet (อัปเดตยอดเงิน)
    ↓
Transactions Sheet (บันทึก)
    ↓
UsersBalance Sheet (อัปเดต)
```

### ตัวอย่าง: เติมเงิน 500 บาท

**ก่อนเติมเงิน:**
```
Players Sheet:
| User ID | ชื่อ LINE | ยอดเงิน |
| U2fbb940e366149fc8dbdf92b92a81b60 | 💓Noon💓 | 5000 |

UsersBalance Sheet:
| User ID | ชื่อ LINE | ยอดเงิน |
| U2fbb940e366149fc8dbdf92b92a81b60 | 💓Noon💓 | 5000 |
```

**หลังเติมเงิน:**
```
Players Sheet:
| User ID | ชื่อ LINE | ยอดเงิน |
| U2fbb940e366149fc8dbdf92b92a81b60 | 💓Noon💓 | 5500 |

Transactions Sheet (เพิ่มแถวใหม่):
```

## 🎯 สรุป

| Sheet | ใช้งาน | อัปเดต | ความสำคัญ |
|-------|--------|--------|---------|
| Players | ข้อมูลผู้เล่น + ยอดเงิน | เติมเงิน, ประกาศผล | ⭐⭐⭐ (หลัก) |
| Transactions | บันทึกการเปลี่ยนแปลง | เติมเงิน, ประกาศผล | ⭐⭐⭐ (Audit) |
| UsersBalance | ยอดเงินปัจจุบัน | อัตโนมัติจาก Players | ⭐⭐ (ค้นหาเร็ว) |
| Bets | บันทึกการเดิมพัน | จับคู่เล่น, ประกาศผล | ⭐⭐⭐ (บันทึก) |

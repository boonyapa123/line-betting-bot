# รูปแบบการเล่นใหม่: ไล่/350-390/10000แอดไล่

## สรุปการเปลี่ยนแปลง

เพิ่มรูปแบบการเล่นใหม่ที่รองรับรูปแบบ Slash: `[ประเภท]/[ช่วงราคา]/[ยอดเงิน][ชื่อบั้งไฟ]`

### ตัวอย่าง
- `ไล่/350-390/10000แอดไล่` = ประเภท: ไล่ (สูง), ช่วง: 350-390, ยอดเงิน: 10000, ชื่อบั้งไฟ: แอดไล่
- `ต/360-410/5000แอดต` = ประเภท: ต (ต่ำ), ช่วง: 360-410, ยอดเงิน: 5000, ชื่อบั้งไฟ: แอดต
- `ย/370-400/8000แอดย` = ประเภท: ย (ยั้ง), ช่วง: 370-400, ยอดเงิน: 8000, ชื่อบั้งไฟ: แอดย
- `ส/380-420/12000แอดส` = ประเภท: ส (สูง), ช่วง: 380-420, ยอดเงิน: 12000, ชื่อบั้งไฟ: แอดส

## ไฟล์ที่ปรับปรุง

### 1. services/betting/bettingMessageParserService.js
- เพิ่ม regex pattern: `METHOD2_SLASH_PATTERN`
- เพิ่มฟังก์ชัน: `parseMethod2Slash()`
- ปรับปรุง `parseMessage()` เพื่อตรวจสอบรูปแบบใหม่ก่อน
- ปรับปรุง `getHint()` เพื่อรวมรูปแบบใหม่

### 2. index.js
- ปรับปรุง `extractBetType()` เพื่อรองรับรูปแบบ Slash
- ปรับปรุง `extractBetAmount()` เพื่อรองรับรูปแบบ Slash
- ปรับปรุง `extractPriceRange()` เพื่อรองรับรูปแบบ Slash
- ปรับปรุง `extractFireworkName()` เพื่อรองรับรูปแบบ Slash

## ความแตกต่างระหว่าง Reply vs Auto Matching

| เงื่อนไข | Reply Matching | Auto Matching |
|---------|----------------|---------------|
| **ตรวจชื่อบั้งไฟ** | ❌ ไม่ต้องตรวจ | ✅ ต้องตรงกันทั้งหมด |
| **ประเภทเดิมพัน** | ✅ ต้องตรงข้าม (หรือไม่ระบุ) | ✅ ต้องตรงข้าม |
| **ยอดเงิน** | 💰 ใช้ User A | 💰 ใช้ยอดน้อยกว่า |
| **การทำงาน** | User B reply ข้อความ | ค้นหาอัตโนมัติจากชีท |

## ประเภทเดิมพันที่รองรับ

### ประเภทตรงข้าม
- `ถอย` ↔️ `ยั้ง`
- `ล่าง` ↔️ `บน`
- `ต่ำ/ยั่ง` ↔️ `สูง/ไล่`
- `ชล` ↔️ `ชล` (เดียวกัน)

### ตัวอักษรย่อ
- `ต` = `ต่ำ/ยั่ง`
- `ย` = `ต่ำ/ยั่ง`
- `ส` = `สูง/ไล่`
- `ล` = `สูง/ไล่`
- `ถ` = `ถอย`
- `บ` = `บน`

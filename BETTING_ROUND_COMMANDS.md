# 📋 คำสั่งเปิด/ปิดรอบการเล่น

## 🚀 คำสั่งเปิดรอบการเล่น

### ใช้ Node.js

```bash
node open-betting-round.js "ชื่อบั้งไฟ"
```

### ตัวอย่าง

```bash
# เปิดรอบบั้งไฟ 340-370
node open-betting-round.js "340-370"

# เปิดรอบบั้งไฟ 320-340
node open-betting-round.js "320-340"

# เปิดรอบบั้งไฟ 310-330
node open-betting-round.js "310-330"
```

## 📊 ขั้นตอนการใช้งาน

### 1️⃣ เปิดรอบการเล่น
```bash
node open-betting-round.js "340-370"
```

### 2️⃣ ผู้เล่นส่งข้อความแทง
```
340-370 ย 400 ศ.
340-370 ล 400 ศ.
```

### 3️⃣ ระบบจับคู่อัตโนมัติ
ระบบจะจับคู่เล่นอัตโนมัติและส่งแจ้งเตือน

### 4️⃣ ประกาศผล
ส่งข้อความประกาศผล เช่น:
```
340-370 ออก 365
```

### 5️⃣ ปิดรอบการเล่น
```bash
node close-betting-round.js
```

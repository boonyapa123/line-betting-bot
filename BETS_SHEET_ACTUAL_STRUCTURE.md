# โครงสร้างชีท Bets (จริง) - Actual Bets Sheet Structure

## 📊 ภาพรวม

ชีท **Bets** มีโครงสร้างดังนี้:

```
A | B | C | D | E | F | G | H | I | J | K | L | M | N
```

---

|--------|------|--------|---------|

| **B** | User A ID | ID ผู้เล่น A | U2fbb940e366149fc8dbdf92b92a81b60 |

## 📝 รายละเอียดแต่ละคอลัมน์

### A: Timestamp (วันเวลา)
**ทำหน้าที่:** บันทึกเวลาที่ผู้เล่นเล่น

**ข้อมูล:** ISO 8601 format
```
2024-03-02T14:56:01.477Z
```

---

### B: User A ID (ID ผู้เล่น A)
**ทำหน้าที่:** ระบุตัวตนของผู้เล่น A

**ข้อมูล:** LINE User ID
```
U2fbb940e366149fc8dbdf92b92a81b60
```

---

### C: ชื่อ User A (ชื่อผู้เล่น A)
**ทำหน้าที่:** แสดงชื่อผู้เล่น A

**ข้อมูล:** ชื่อที่ LINE แสดง
```
💓Noon💓
```

---

### D: ข้อความ A (ข้อความที่ส่ง)
**ทำหน้าที่:** บันทึกข้อความที่ผู้เล่น A ส่ง

**ข้อมูล:** ข้อความเต็ม
```
ลูกชายภูน้อย ชล 150
```

---

### E: ชื่อบั้งไฟ (ชื่อบั้งไฟ)
**ทำหน้าที่:** ระบุบั้งไฟที่เล่น

**ข้อมูล:** ชื่อบั้งไฟ
```
ลูกชายภูน้อย
```

---

### F: รายการเล่น (ฝั่ง + จำนวนเงิน)
**ทำหน้าที่:** บันทึกรายการเล่น

**ข้อมูล:** ฝั่ง + จำนวนเงิน
```
ชล 150
ชถ 300
```

---

### G: ยอดเงิน (จำนวนเงิน User A)
**ทำหน้าที่:** บันทึกจำนวนเงินที่ User A เล่น

**ข้อมูล:** ตัวเลข (บาท)
```
150
500
```

---

### H: ยอดเงิน B (จำนวนเงิน User B)
**ทำหน้าที่:** บันทึกจำนวนเงินที่ User B เล่น

**ข้อมูล:** ตัวเลข (บาท) หรือ ว่างเปล่า (รอการจับคู่)
```
150
300
(ว่างเปล่า - รอการจับคู่)
```

---

### I: แสดงผล (ผลลัพธ์)
**ทำหน้าที่:** บันทึกผลลัพธ์

**ข้อมูล:** ชนะ/แพ้
```
ชนะ
แพ้
```

---

### J: แสดงผลชนะ (ผลลัพธ์ชนะ)
**ทำหน้าที่:** บันทึกจำนวนเงินที่ชนะ

**ข้อมูล:** ตัวเลข (บาท)
```
450
-150
```

---

### K: User B ID (ID ผู้เล่น B)
**ทำหน้าที่:** ระบุตัวตนของผู้เล่น B

**ข้อมูล:** LINE User ID หรือ ว่างเปล่า (รอการจับคู่)
```
Ubb4a196cd0dc0c3ca6211a9bafecb513
(ว่างเปล่า - รอการจับคู่)
```

---

### L: ชื่อ User B (ชื่อผู้เล่น B)
**ทำหน้าที่:** แสดงชื่อผู้เล่น B

**ข้อมูล:** ชื่อที่ LINE แสดง หรือ ว่างเปล่า (รอการจับคู่)
```
Pam Yuthida
(ว่างเปล่า - รอการจับคู่)
```

---

### M: รายการเล่น B (ฝั่ง + จำนวนเงิน B)
**ทำหน้าที่:** บันทึกรายการเล่นของ User B

**ข้อมูล:** ฝั่ง + จำนวนเงิน หรือ ว่างเปล่า (รอการจับคู่)
```
ชถ 150
(ว่างเปล่า - รอการจับคู่)
```

---

### N: ผลลัพธ์สุดท้าย (ผลลัพธ์สุดท้าย)
**ทำหน้าที่:** บันทึกผลลัพธ์สุดท้าย

**ข้อมูล:** ชนะ/แพ้ หรือ ว่างเปล่า (รอการจับคู่)
```
ชนะ
แพ้
(ว่างเปล่า - รอการจับคู่)
```

---

## 📋 ตัวอย่างข้อมูลในชีท Bets

### ตัวอย่าง 1: ผู้เล่นลงเล่นตรง (รอการจับคู่)

```
| A | B | C | D | E | F | G | H | I | J | K | L | M | N |
|---|---|---|---|---|---|---|---|---|---|---|---|---|---|
| 2024-03-02T14:56:01.477Z | U2fbb940e366149fc8dbdf92b92a81b60 | 💓Noon💓 | ลูกชายภูน้อย ชล 150 | ลูกชายภูน้อย | ชล 150 | 150 | | | | | | | |
```

**ความหมาย:**
- User A (💓Noon💓) ลงเล่น ลูกชายภูน้อย ชล 150 บาท
- รอการจับคู่ (H-N ว่างเปล่า)

---

### ตัวอย่าง 2: จับคู่สำเร็จ

```
| A | B | C | D | E | F | G | H | I | J | K | L | M | N |
|---|---|---|---|---|---|---|---|---|---|---|---|---|---|
| 2024-03-02T14:56:01.477Z | U2fbb940e366149fc8dbdf92b92a81b60 | 💓Noon💓 | ลูกชายภูน้อย ชล 150 | ลูกชายภูน้อย | ชล 150 | 150 | 150 | | | Ubb4a196cd0dc0c3ca6211a9bafecb513 | Pam Yuthida | ชถ 150 | |
```

**ความหมาย:**
- User A (💓Noon💓) ลงเล่น ชล 150 บาท
- User B (Pam Yuthida) ตอบกลับ ชถ 150 บาท
- จับคู่สำเร็จ (รอสรุปผล)

---

### ตัวอย่าง 3: สรุปผลแล้ว (User A ชนะ)

```
| A | B | C | D | E | F | G | H | I | J | K | L | M | N |
|---|---|---|---|---|---|---|---|---|---|---|---|---|---|
| 2024-03-02T14:56:01.477Z | U2fbb940e366149fc8dbdf92b92a81b60 | 💓Noon💓 | ลูกชายภูน้อย ชล 150 | ลูกชายภูน้อย | ชล 150 | 150 | 150 | ชนะ | 450 | Ubb4a196cd0dc0c3ca6211a9bafecb513 | Pam Yuthida | ชถ 150 | ชนะ |
```

**ความหมาย:**
- User A ชนะ ได้ 450 บาท
- User B แพ้ เสีย 150 บาท

# ระบบบันทึกผลลัพธ์การเล่น - อัปเดต

## 📋 สรุปการเปลี่ยนแปลง

ระบบได้รับการปรับปรุงให้สามารถ:
1. ✅ ระบุได้ว่าใครเล่นกับใคร ใครแพ้ ใครชนะ
2. ✅ คำนวณค่าธรรมเนียม (หัก 10% ยอดเล่น, หัก 5% ออกกลาง)
3. ✅ แจ้งเตือน LINE ส่วนตัวและกลุ่ม
4. ✅ บันทึกชื่อ LINE ของผู้เล่น

---

## 🔧 ไฟล์ที่เพิ่มเติม

### 1. `services/betting/bettingResultService.js`
บริหารจัดการผลลัพธ์การเล่น คำนวณค่าธรรมเนียม และแจ้งเตือน

**ฟังก์ชันหลัก:**
- `calculateResultWithFees()` - คำนวณผลลัพธ์พร้อมค่าธรรมเนียม
- `recordResult()` - บันทึกผลลัพธ์ลงชีท Results
- `notifyLineResult()` - แจ้งเตือน LINE ส่วนตัวและกลุ่ม
- `buildResultMessage()` - สร้างข้อความผลลัพธ์

**ค่าธรรมเนียม:**
- ชนะ-แพ้: หัก 10% จากยอดเล่น
- ออกกลาง: หัก 5% ทั้งสองฝั่ง

### 2. `services/line/lineNotificationService.js`
ส่งข้อความแจ้งเตือนไปยัง LINE

**ฟังก์ชันหลัก:**
- `sendPrivateMessage()` - ส่งข้อความส่วนตัว
- `sendGroupMessage()` - ส่งข้อความกลุ่ม
- `sendReplyMessage()` - ส่งข้อความตอบกลับ
- `sendFlexMessage()` - ส่ง Flex Message

### 3. `scripts/update-bets-sheet-columns.js`
อัปเดตชีท Google Sheets

## 🔄 ขั้นตอนการทำงาน

### 1. ผู้เล่นส่งข้อความเล่น
```
User: "ต 100"
```

### 2. ระบบบันทึกการเล่น
- บันทึกลงชีท "Bets" พร้อมชื่อ LINE
- ส่งข้อความยืนยัน

### 3. Admin สรุปผล
```
POST /betting/admin/calculate
{
  "slipName": "บั้งไฟ1",
  "score": 50
}
```

### 4. ระบบคำนวณผลลัพธ์
- จับคู่การเล่น
- คำนวณค่าธรรมเนียม
- บันทึกลงชีท "Results"
- อัปเดตยอดเงิน

### 5. แจ้งเตือน LINE
- **ส่วนตัวผู้ชนะ:** ยินดีด้วย + ยอดได้รับ
- **ส่วนตัวผู้แพ้:** เสียใจด้วย + ยอดเสีย
- **กลุ่ม:** ผลลัพธ์ทั้งหมด

## 💰 ตัวอย่างการคำนวณ

### ชนะ-แพ้ (หัก 10%)
```
ผู้เล่น A เดิมพัน: 100 บาท
ผู้เล่น B เดิมพัน: 100 บาท

ผู้ชนะ (A):
- เดิมพัน: 100 บาท
- ค่าธรรมเนียม: 10 บาท (10%)
- ได้รับ: 90 บาท

ผู้แพ้ (B):
- เดิมพัน: -100 บาท
```

### ออกกลาง (หัก 5%)
```
ผู้เล่น A เดิมพัน: 100 บาท
ผู้เล่น B เดิมพัน: 100 บาท

ทั้งสองฝั่ง:
- ค่าธรรมเนียม: 5 บาท (5%)
- ได้รับ: -5 บาท
```

## 📨 ประเภทข้อความที่ส่ง

### 1️⃣ ข้อความยืนยันการเล่น (Confirmation)

**เมื่อ:** User ส่งข้อความเล่นถูกรูปแบบ

**ข้อความ:**
```
✅ บันทึกการเล่นสำเร็จ

ชื่อ: Alice
บั้งไฟ: ฟ้าหลังฝน
ฝั่ง: ไล่
จำนวนเงิน: 500 บาท
```

**Code:**
```javascript
buildConfirmationMessage(parsedBet, displayName) {
  let message = `✅ บันทึกการเล่นสำเร็จ\n\n`;
  message += `ชื่อ: ${displayName}\n`;
  message += `บั้งไฟ: ${parsedBet.slipName}\n`;
  
  if (parsedBet.method === 1) {
    message += `ฝั่ง: ${parsedBet.side}\n`;
  } else {
    message += `ราคา: ${parsedBet.price}\n`;
    message += `ฝั่ง: ${parsedBet.side}\n`;
  }
  
  message += `จำนวนเงิน: ${parsedBet.amount} บาท`;
  return message;
}
```

---

### 2️⃣ ข้อความข้อผิดพลาด (Error)

**เมื่อ:** ข้อความไม่ตรงรูปแบบ

**ข้อความ:**
```
❌ รูปแบบผิดครับ กรุณาตรวจสอบการเว้นวรรค

วิธีที่ 1: [ชื่อบั้งไฟ] [ชล./ชถ.] [ยอดเงิน]
ตัวอย่าง: ฟ้าหลังฝน ชล. 500

วิธีที่ 2: [ราคา] [ล./ย.] [ยอดเงิน] [ชื่อบั้งไฟ]
ตัวอย่าง: 0/3(300-330) ล. 500 ฟ้าหลังฝน
```

**ตัวอย่างข้อผิดพลาดอื่นๆ:**
```
❌ จำนวนเงินต้องมากกว่า 0
❌ จำนวนเงินเกินขีดจำกัด
❌ ชื่อบั้งไฟไม่ถูกต้อง
❌ รอบนี้ปิดการทายแล้ว
```

---

### 3️⃣ ข้อความเปิดรอบ (Start Round)

**เมื่อ:** Admin ส่ง `:เริ่ม [ชื่อบั้งไฟ]`

**ข้อความ:**
```
✅ เปิดรอบการเล่น: ฟ้าหลังฝน
รอบ ID: ROUND_1709443800000
```

**Code:**
```javascript
async handleStartCommand(slipName) {
  const result = await bettingRoundStateService.openRound(slipName);
  
  if (result.success) {
    return {
      type: 'text',
      text: `✅ ${result.message}\nรอบ ID: ${result.roundId}`,
    };
  }
}
```

---

### 4️⃣ ข้อความปิดรอบ (Stop Round)

**เมื่อ:** Admin ส่ง `:หยุด`

**ข้อความ:**
```
รอบนี้ปิดการทายแล้วคะ/ครับ
```

**Code:**
```javascript
async handleStopCommand() {
  const result = await bettingRoundStateService.closeRound();
  
  if (result.success) {
    return {
      type: 'text',
      text: result.message,
    };
  }
}
```

---

### 5️⃣ ข้อความประกาศผลลัพธ์ (Result Announcement)

**เมื่อ:** Admin ส่ง `:สรุป [ชื่อบั้งไฟ] [คะแนน]`

**ข้อความ:**
```
📊 สรุปผลการเล่น
บั้งไฟ: ฟ้าหลังฝน
คะแนนที่ออก: 315
========================================

คู่ที่ 1:
🏆 ชนะ: Alice +500 บาท
❌ แพ้: Bob -500 บาท

คู่ที่ 2:
🏆 ชนะ: Charlie +1000 บาท
❌ แพ้: David -1000 บาท

========================================
💰 ยอดเงินคงเหลือ:

Alice: 1500 บาท
Bob: 500 บาท
Charlie: 2000 บาท
David: 1000 บาท
```

**Code:**
```javascript
async buildResultReport(slipName, score, results) {
  let report = `📊 สรุปผลการเล่น\n`;
  report += `บั้งไฟ: ${slipName}\n`;
  report += `คะแนนที่ออก: ${score}\n`;
  report += `${'='.repeat(40)}\n\n`;

  for (let i = 0; i < results.length; i++) {
    const result = results[i];
    report += `คู่ที่ ${i + 1}:\n`;
    report += `🏆 ชนะ: ${result.winner.displayName} +${result.winner.amount} บาท\n`;
    report += `❌ แพ้: ${result.loser.displayName} -${result.loser.amount} บาท\n\n`;
  }

  const balances = await bettingPairingService.getAllBalances();
  report += `${'='.repeat(40)}\n`;
  report += `💰 ยอดเงินคงเหลือ:\n\n`;

  for (const balance of balances) {
    report += `${balance.displayName}: ${balance.balance} บาท\n`;
  }

  return report;
}
```

## 🎯 สรุปข้อความที่ส่ง

| ประเภท | เมื่อ | ข้อความ |
|--------|------|--------|
| ✅ Confirmation | User เล่นถูก | บันทึกการเล่นสำเร็จ |
| ❌ Error | ข้อมูลผิด | รูปแบบผิด / จำนวนเงินผิด |
| ✅ Start | Admin :เริ่ม | เปิดรอบการเล่น |
| 🛑 Stop | Admin :หยุด | ปิดการทายแล้ว |
| 📊 Result | Admin :สรุป | สรุปผลการเล่น + ยอดเงิน |

## 💡 ข้อสำคัญ

✅ **ข้อความส่งเข้าห้องแชทเดียวกัน**
- ผู้เล่นทั้งหมดในห้องแชทจะเห็นข้อความ
- ไม่มีข้อความส่วนตัว

✅ **ข้อความส่งทันที**
- ไม่มี delay
- ผู้เล่นได้รับข้อมูลแบบ real-time

✅ **ข้อความมีรายละเอียด**
- ชื่อ User
- ชื่อบั้งไฟ
- ฝั่งที่เล่น
- จำนวนเงิน
- ผลลัพธ์
- ยอดเงินคงเหลือ

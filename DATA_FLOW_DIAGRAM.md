## 📊 Data Structure

```json
{
  "events": [
    {
      "type": "message",
      "source": {
        "userId": "U1234567890abcdef1234567890abcdef",
        "groupId": "C1234567890abcdef1234567890abcdef"
      },
      "message": {
        "type": "image",
        "id": "100001"
      }
    }
  ]
}
```

### 2. QR Code String
```
0041000600000101030040220014242082547BPM049885102TH9104xxxx
```

```json
{
  "payload": {
```

## 🔄 Data Transformation

```
Image Buffer
    ↓
    ├─ Jimp (อ่านรูปภาพ)
    ├─ ImageData (แปลงเป็น ImageData)
    └─ jsQR (สแกน QR Code)
    ↓
QR Code String
    ↓
    ├─ Slip2Go API (ตรวจสอบ)
    └─ Slip Data (ได้ข้อมูลสลิป)
    ↓
Slip Data
    ├─ Google Sheets (บันทึก)
    └─ LINE Message (สร้างข้อความ)
    ↓
Response Message
    ↓
    └─ LINE API (ส่งข้อความ)
    ↓
User Message
```

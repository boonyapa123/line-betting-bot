# 📊 ระบบประกาศผลเดิมพัน - อัปเดต

## ✅ การเปลี่ยนแปลง

### 1️⃣ บันทึกผลลัพธ์ลงชีท Bets

**ฟังก์ชัน**: `recordResult()` ใน `bettingResultService.js`

**ทำการ:**
- ✅ ค้นหาแถวที่จับคู่ไปแล้ว (User A + User B)
- ✅ อัปเดต Column I: ผลที่ออก (คะแนน)
- ✅ อัปเดต Column J: ผลแพ้ชนะ (ชนะ/แพ้/เสมอ)
- ✅ อัปเดต Column S: ผลลัพธ์ A (ชนะ/แพ้ + จำนวนเงิน)
- ✅ อัปเดต Column T: ผลลัพธ์ B (ชนะ/แพ้ + จำนวนเงิน)

**ตัวอย่าง:**
```
Row 2 (User A + User B):
  Column I: 340 (ผลที่ออก)
  Column J: ชนะ (ผลแพ้ชนะ)
  Column S: ชนะ 270 บาท (ผลลัพธ์ A)
  Column T: แพ้ 300 บาท (ผลลัพธ์ B)
```

---

### 2️⃣ ประกาศผลด้วยรูปแบบใหม่

**ฟังก์ชัน**: `buildResultMessage()` ใน `bettingResultService.js`

**รูปแบบใหม่:**
```
แอด 460✅️เทวดา 300❌️เทพพนม 350⛔️ฟ้าหลังฝน
```

**ความหมาย:**
- ✅️ = ชนะ
- ❌️ = แพ้
- ⛔️ = เสมอ (จาว)

**ตัวอย่าง:**

#### ชนะ-แพ้
```
แอด 460✅️เทวดา 300❌️เทพพนม
```
- บั้งไฟ: แอด
- คะแนน: 460
- ผู้ชนะ: เทวดา (ได้ 300 บาท)
- ผู้แพ้: เทพพนม

#### ออกกลาง (เสมอ)
```
แอด 350⛔️เทวดา ⛔️เทพพนม
```
- บั้งไฟ: แอด
- คะแนน: 350
- ผลลัพธ์: เสมอ (ทั้งสองฝั่ง)

---

### 3️⃣ ส่งแจ้งเตือนด้วย Account Line OA ของกลุ่ม

**ขั้นตอน:**
1. ดึง `groupId` จาก source
2. ค้นหา Account Number ของกลุ่มจาก `data/groups.json`
3. ใช้ Account นั้นส่งแจ้งเตือน

**ฟังก์ชัน**: `notifyLineResult()` ใน `bettingResultService.js`

**ส่งแจ้งเตือน 3 ประเภท:**
1. ✅ ข้อความส่วนตัวให้ผู้ชนะ
2. ✅ ข้อความส่วนตัวให้ผู้แพ้
3. ✅ ข้อความกลุ่ม (ใช้ Account ของกลุ่ม)

---

## 📝 ตัวอย่างข้อความแจ้งเตือน

### ผู้ชนะ (ส่วนตัว)
```
🎉 ยินดีด้วย! คุณชนะ

บั้งไฟ: แอด
คะแนนที่ออก: 460

✅️ ชนะ
เดิมพัน: 300 บาท
หัก: 30 บาท (10%)
ได้รับ: 270 บาท
```

### ผู้แพ้ (ส่วนตัว)
```
😔 เสียใจด้วย คุณแพ้

บั้งไฟ: แอด
คะแนนที่ออก: 460

❌️ แพ้
เดิมพัน: 300 บาท
```

### กลุ่ม
```
แอด 460✅️เทวดา 300❌️เทพพนม
```

## 🎯 ขั้นตอนการประกาศผล

```
Admin: ":สรุป แอด 460"
  ↓
[Step 1] Parse Command
  - slipName: "แอด"
  - score: 460
  ↓
[Step 2] handleCalculateCommand(slipName, score, source)
  - ดึง groupId จาก source
  - ดึง Account Number ของกลุ่ม
  ↓
[Step 3] จับคู่การเล่น
  ↓
[Step 4] คำนวณผลลัพธ์
  ↓
[Step 5] บันทึกผลลัพธ์ลงชีท Bets
  - Column I, J, S, T
  ↓
[Step 6] อัปเดตยอดเงิน
  ↓
[Step 7] แจ้งเตือน LINE
  - ส่วนตัว: ผู้ชนะ + ผู้แพ้
  - กลุ่ม: "แอด 460✅️เทวดา 300❌️เทพพนม"
  ↓
[Step 8] ล้างข้อมูล + ปิดรอบ
```

---

## 📌 หมายเหตุ

- ระบบจะใช้ Account Line OA ของกลุ่มในการส่งแจ้งเตือน
- ถ้าไม่พบ Account ของกลุ่ม จะใช้ Account 1 เป็น default
- ผลลัพธ์จะบันทึกลงชีท Bets ทันทีหลังจากประกาศผล

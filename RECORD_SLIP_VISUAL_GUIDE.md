# 📊 การบันทึกข้อมูลสลิป - Visual Guide

## 🎯 ภาพรวม

```
ผู้ใช้ส่งรูปภาพสลิป
    ↓
Server ตรวจสอบสลิป
    ↓
สลิปถูกต้อง? ✅
    ↓
บันทึกลง Google Sheets
    ↓
Google Sheets
```

## 📋 ข้อมูลที่บันทึก

### Worksheet: "Slip Verification"

```
┌──────────────┬──────────────┬──────────────┬──────────┬──────────────┬──────────────┐
│ วันที่บันทึก │ Reference ID │ Transaction  │ จำนวนเงิน │ วันที่โอน    │ ชื่อผู้ส่ง    │
├──────────────┼──────────────┼──────────────┼──────────┼──────────────┼──────────────┤
│ 26/02/2025   │ 92887bd5-... │ 015073144... │ 1000     │ 2025-10-05   │ สมชาย        │
│ 26/02/2025   │ 92887bd6-... │ 015073144... │ 2000     │ 2025-10-05   │ สมชาย        │
│ 26/02/2025   │ 92887bd7-... │ 015073144... │ 500      │ 2025-10-05   │ สมชาย        │
└──────────────┴──────────────┴──────────────┴──────────┴──────────────┴──────────────┘
```

### Header (แถวแรก)

```
A: วันที่บันทึก
B: Reference ID
C: Transaction Ref
D: จำนวนเงิน
E: วันที่โอน
F: ชื่อผู้ส่ง
G: บัญชีผู้ส่ง
H: ธนาคารผู้ส่ง
I: ชื่อผู้รับ
J: บัญชีผู้รับ
K: ธนาคารผู้รับ
L: สถานะ
```

## 🔄 ขั้นตอนการบันทึก

### Step 1: ผู้ใช้ส่งรูปภาพสลิป

```
┌─────────────────────────────────────────┐
│  ผู้ใช้                                  │
├─────────────────────────────────────────┤
│                                         │
│  เปิด LINE OA                           │
│  ↓                                      │
│  ส่งรูปภาพสลิป                          │
│  ↓                                      │
│  LINE OA รับรูปภาพ                      │
│                                         │
└─────────────────────────────────────────┘
```

### Step 2: Server ตรวจสอบสลิป

```
┌─────────────────────────────────────────┐
│  Server                                 │
├─────────────────────────────────────────┤
│                                         │
│  1. สแกน QR Code ✅                     │
│  2. เรียก Slip2Go API ✅                │
│  3. ได้ผลลัพธ์ ✅                       │
│                                         │
│  ผลลัพธ์:                               │
│  {                                      │
│    "code": "200000",                    │
│    "amount": 1000,                      │
│    "sender": "สมชาย",                   │
│    "receiver": "บริษัท"                 │
│  }                                      │
│                                         │
└─────────────────────────────────────────┘
```

### Step 3: ตรวจสอบว่าสลิปถูกต้อง

```
┌─────────────────────────────────────────┐
│  ตรวจสอบผลลัพธ์                         │
├─────────────────────────────────────────┤
│                                         │
│  code = "200000"?                       │
│  ├─ ใช่ → บันทึก ✅                     │
│  └─ ไม่ → ไม่บันทึก ❌                  │
│                                         │
└─────────────────────────────────────────┘
```

### Step 4: ตรวจสอบสลิปซ้ำ

```
┌─────────────────────────────────────────┐
│  SlipRecordingService                   │
├─────────────────────────────────────────┤
│                                         │
│  ค้นหา referenceId ใน Google Sheets     │
│  ↓                                      │
│  พบ?                                    │
│  ├─ ใช่ → สลิปซ้ำ ❌ (ไม่บันทึก)        │
│  └─ ไม่ → สลิปใหม่ ✅ (บันทึก)          │
│                                         │
└─────────────────────────────────────────┘
```

### Step 5: บันทึกข้อมูลลง Google Sheets

```
┌─────────────────────────────────────────┐
│  SlipRecordingService.recordSlip()      │
├─────────────────────────────────────────┤
│                                         │
│  1. ตรวจสอบ Worksheet                  │
│     ├─ มีอยู่? → ใช้                    │
│     └─ ไม่มี? → สร้างใหม่               │
│                                         │
│  2. เพิ่ม Header (ถ้าเป็นครั้งแรก)     │
│     ├─ วันที่บันทึก                     │
│     ├─ Reference ID                     │
│     ├─ Transaction Ref                  │
│     └─ ... (ทั้งหมด 12 columns)        │
│                                         │
│  3. บันทึกข้อมูลแถวใหม่                 │
│     ├─ 26/02/2025 14:30:45             │
│     ├─ 92887bd5-...                    │
│     ├─ 015073144...                    │
│     └─ ... (ทั้งหมด 12 columns)        │
│                                         │
│  4. Google Sheets API                   │
│     ├─ Append values to sheet           │
│     └─ ✅ บันทึกสำเร็จ                  │
│                                         │
└─────────────────────────────────────────┘
```

## 🔐 ตั้งค่า Google Sheets

### Step 1: สร้าง Google Sheets

```
┌─────────────────────────────────────────┐
│  Google Sheets                          │
├─────────────────────────────────────────┤
│                                         │
│  1. ไปที่ https://sheets.google.com     │
│  2. สร้าง Spreadsheet ใหม่              │
│  3. ตั้งชื่อ                            │
│     "Slip Verification Records"         │
│  4. คัดลอก Sheet ID จาก URL            │
│                                         │
│  URL: https://docs.google.com/...      │
│       /d/[SHEET_ID]/edit                │
│                                         │
└─────────────────────────────────────────┘
```

### Step 2: ตั้งค่า Service Account

```
┌─────────────────────────────────────────┐
│  Google Cloud Console                   │
├─────────────────────────────────────────┤

│  3. ดาวน์โหลด JSON credentials         │
│  4. บันทึกไว้เป็น credentials.json      │
│                                         │
└─────────────────────────────────────────┘
```

### Step 3: แชร์ Google Sheets

```
┌─────────────────────────────────────────┐

## 📊 Timeline

```
⏰ 0s   - ผู้ใช้ส่งรูปภาพสลิป

⏰ 3s   - ดาวน์โหลดรูปภาพ

# 📢 ระบบแจ้งผลการเล่น

## 📋 ภาพรวม

เมื่อประกาศผลบั้งไฟ ระบบจะ:
1. ✅ อัปเดตผลลัพธ์ในชีท Bets
2. ✅ คำนวนแพ้ชนะ
3. ✅ อัปเดตยอดเงินผู้เล่น
4. ✅ ส่งข้อความแจ้งผลให้ผู้ชนะและผู้แพ้ (ส่วนตัว)
5. ✅ ส่งข้อความแจ้งผลในกลุ่มแชท

---

## 🔄 ขั้นตอนการแจ้งผล

### ขั้นตอนที่ 1: ประกาศผล

```
ผู้ประกาศ → ส่งข้อความผล → LINE Group
            "บั้งไฟ 310 ✅"
```

### ขั้นตอนที่ 2: ตรวจสอบและจับคู่

```
ระบบ → ค้นหาการเดิมพันที่ตรงกับบั้งไฟ 310
    → จับคู่ผู้เล่น (User A vs User B)
    → ดึงข้อมูลจากชีท Bets
```

### ขั้นตอนที่ 3: อัปเดตผลลัพธ์

```
ระบบ → อัปเดตคอลัมน์ I, J, K ในชีท Bets
    ├─ Column I: ผลที่ออก (310)
    ├─ Column J: ผลแพ้ชนะ User A (✅)
    └─ Column K: ผลแพ้ชนะ User B (❌)
```

### ขั้นตอนที่ 4: คำนวนแพ้ชนะ

```
ถ้า User A ชนะ (✅):
  - User A ได้รับ: เดิมพัน - 10% commission
  - User B เสีย: -เดิมพัน

ถ้า User A แพ้ (❌):
  - User A เสีย: -เดิมพัน
  - User B ได้รับ: เดิมพัน - 10% commission

ถ้าเสมอ (⛔️):
  - User A เสีย: -5% commission
  - User B เสีย: -5% commission
```

### ขั้นตอนที่ 5: อัปเดตยอดเงิน

```
ระบบ → อัปเดตยอดเงินใน Players Sheet
    ├─ User A: ยอดเงินเก่า + แพ้ชนะ
    └─ User B: ยอดเงินเก่า + แพ้ชนะ
```

### ขั้นตอนที่ 6: ส่งข้อความแจ้งผล

```
ระบบ → ส่งข้อความส่วนตัวให้ผู้ชนะ
    ├─ ✅ ชนะแล้ว
    ├─ บั้งไฟ, เดิมพัน, ได้รับ
    ├─ ยอดคงเหลือ
    └─ ชื่อผู้แพ้

ระบบ → ส่งข้อความส่วนตัวให้ผู้แพ้
    ├─ ❌ แพ้แล้ว
    ├─ บั้งไฟ, เดิมพัน, เสีย
    ├─ ยอดคงเหลือ
    └─ ชื่อผู้ชนะ

ระบบ → ส่งข้อความในกลุ่มแชท
    ├─ ผลการเล่นทั้งหมด
    ├─ ชื่อผู้ชนะ/แพ้
    └─ จำนวนเงิน
```

## 🔍 ตัวอย่างการทำงาน

### ตัวอย่าง 1: User A ชนะ

```
ประกาศผล: "บั้งไฟ 310 ✅"

ขั้นตอน:
1. ค้นหาการเดิมพัน → พบ 2 คู่
2. อัปเดตผลลัพธ์:
   - Result Number: 310
   - Result A: ✅
   - Result B: ❌
3. คำนวนแพ้ชนะ:
   - User A (สมชาย): 100 - 10 = 90 บาท
   - User B (วิชัย): -100 บาท
4. อัปเดตยอดเงิน:
   - User A: 500 + 90 = 590 บาท
   - User B: 600 - 100 = 500 บาท
5. ส่งข้อความ:
   - User A: ✅ ชนะแล้ว | ได้รับ: 90 บาท | ยอดคงเหลือ: 590 บาท
   - User B: ❌ แพ้แล้ว | เสีย: 100 บาท | ยอดคงเหลือ: 500 บาท
   - Group: ผลการเล่น
```

## 🔧 ฟังก์ชันหลัก

### updateBetResult()
- อัปเดตผลลัพธ์ในชีท Bets
- คำนวนแพ้ชนะ
- อัปเดตยอดเงิน
- ส่งข้อความแจ้งผล

### updatePlayerBalance()
- อัปเดตยอดเงินผู้เล่น
- ค้นหาผู้เล่นจากชื่อ LINE (primary)
- ค้นหาผู้เล่นจาก User ID (backup)

### calculateWinnings()
- คำนวนแพ้ชนะ
- หัก commission (10% ชนะ, 5% เสมอ)

## 🐛 Troubleshooting

### ปัญหา: ไม่ส่งข้อความให้ผู้แพ้

**วิธีแก้:**
1. ตรวจสอบว่า User B Name ถูกต้อง (Column M)
2. ตรวจสอบว่า User B ID มีค่า (Column L)
3. ตรวจสอบบันทึก error message

### ปัญหา: ยอดเงินไม่ถูกต้อง

**วิธีแก้:**
1. ตรวจสอบการคำนวน commission
2. ตรวจสอบว่าดึงยอดเงินปัจจุบันถูกต้อง
3. ตรวจสอบบันทึก balance update

### ปัญหา: ไม่แจ้งผลในกลุ่มแชท

**วิธีแก้:**
1. ตรวจสอบว่า Group Name มีค่า (Column O)
2. ตรวจสอบว่า Access Token ถูกต้อง
3. ตรวจสอบบันทึก error message
